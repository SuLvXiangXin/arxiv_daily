Whole-body Motion Control of an Omnidirectional Wheel-Legged Mobile Manipulator via Contact-Aware Dynamic Optimization

arXiv: 2509.14010 作者: Chen, Zong, Li, Shaoyang, Liu, Ben, Li, Min, Yin, Zhouping, Li, Yiqun 日期: 2025/09/17

📝 详细解读

研究背景与动机

轮腿式机器人结合了轮式移动的高效性与腿式移动的强地形适应性，在物流、工业自动化等领域前景广阔。然而，集成机械臂后，系统自由度冗余、轮地接触动力学复杂、移动与操作需协调，使得统一控制极具挑战。当前主流方法主要分为基于模型的控制（如模型预测控制MPC、全身控制WBC）和数据驱动的学习控制（如强化学习RL）。前者通常简化接触模型，例如将轮地线接触近似为点接触以提升计算效率，但牺牲了建模精度；后者则面临仿真到现实差距以及跨平台泛化和高精度操作的困难。此外，对于具有独立转向执行器的全向轮腿平台，现有方法常依赖多种运动模式（如阿克曼转向、蟹行）间的切换，导致控制逻辑复杂、不连续。

本文针对上述痛点，提出了一种新的视角：为具有独立转向执行器的全向轮腿移动操作臂，设计一个统一的、接触感知的全身动态优化控制框架。其核心思路是：构建一个统一的多体动力学模型，同时整合机械臂末端的点接触模型和轮地相互作用的线接触模型，并引入热启动策略加速高维在线优化，从而实现无需模式切换的实时、一致且鲁棒的全身运动控制。

方法详解

本文提出的方法是一个分层的控制框架，其核心在于一个基于接触动力学的全身运动规划与优化控制器。

图8：所提出的全身运动控制框架概览。上层为基于优化的运动规划器，生成参考轨迹和接触力；下层为全身控制器，结合动力学模型和接触约束计算关节指令。热启动策略将上层的优化解作为下层求解器的初始猜测，加速收敛。

1. 平台设计
首先，论文设计并实现了一个全向轮腿式机器人平台。该平台在传统轮腿结构基础上，为每条腿末端增加了独立的360度转向执行器，结合轮毂驱动轮，构成了4WIS-4WID（四轮独立转向-四轮独立驱动）驱动方案。此外，还集成了一个7自由度的灵巧机械臂。这种设计旨在平坦结构化环境中实现高机动性（如零半径转向、蟹行），同时保留一定的地形穿越能力。

图2：四足动物（左）与所提出的全向轮腿机器人（右）的仿生设计对比。机器人每条腿末端配备独立转向模块和驱动轮，模仿了动物的移动、感知和操作能力。

2. 全身多体动力学与接触建模
为了统一控制移动基座和机械臂，论文将整个系统建模为一个单一的多体结构。广义位置 q 和速度 q̇ 包含浮动基座（位置、姿态）、腿部关节和机械臂关节的状态。系统动力学方程为：
M(q)q̈ + h(q, q̇) = Sᵀu + J_c(q)ᵀf_c
其中 M 为惯性矩阵，h 为科里奥利力、离心力和重力项，u 为关节力矩，f_c 为接触力，J_c 为接触点雅可比矩阵。

接触约束要求接触点加速度为零：J_c(q)q̈ + J̇_c(q, q̇)q̇ = 0。结合动力学方程，得到一个包含接触反力的耦合动态约束系统。

3. 统一的接触模型（核心创新点）
本文的关键创新在于对机械臂和轮子采用了差异化的接触建模，并统一在同一个优化框架内：

  • 机械臂末端（点接触）：采用传统的摩擦锥约束，近似为线性接触力锥（CWC），约束形式为 A_c f_c ≤ b_c。
  • 轮地接触（线接触）：采用一种简化的（退化的）六维力约束模型。保留了摩擦锥约束（|f_x| ≤ μ f_z, |f_y| ≤ μ f_z），并对绕x轴（侧倾）和z轴（偏航）的力矩施加约束，但释放了绕y轴（滚动方向）的力矩约束。这既比简单的点接触模型更贴近轮地线接触的物理事实，又比完整的六维力约束更简单，利于实时优化。

图3：坐标系与接触模型示意图。{B}为基座坐标系，{Ee}为末端执行器坐标系，c_i为接触点。黄色锥体表示摩擦锥约束，红色线表示轮地线接触，绿色椭圆共同定义了退化的6维力约束。

为了优化问题的可行性和数值稳定性，将这些线性的不等式硬约束转化为基于残差的软约束，通过惩罚函数 α(r) = 0.5 * rᵀr (if r>0) 的形式纳入目标函数中。

4. 统一的全向运动学模型（另一创新点）
针对4WIS-4WID平台，传统方法需要根据任务在多种运动模式间切换。本文提出了一种几何感知的运动学建模方法，将机器人构型空间描述为 C_sw = SE(2) × (𝕊¹)⁴_steer × (ℝ¹)⁴_wheel，并通过无滑移滚动约束建立机体扭转 ξ 与“形状”速度 u（转向角和轮速）之间的线性关系。该模型能够连续、统一地描述所有运动模式，无需显式的模式切换逻辑。

图4：全向轮腿机器人的四种基本运动模式示意图：(a) 阿克曼转向，(b) 蟹行/对角线运动，(c) 原地旋转，(d) 枢轴转向。所提出的统一运动学模型避免了在这些模式间进行离散切换。

5. 热启动优化策略
为解决高自由度系统在线优化的实时性挑战，论文引入了热启动策略。将低频运行（如100Hz）的上层运动规划器求解出的最优状态轨迹和接触力，作为高频运行（如1kHz）的下层全身控制器优化求解器的初始猜测。这显著加速了求解器的收敛速度，确保了实时控制的可实现性。

总结与启发

核心贡献：

  1. 平台创新：设计并实现了具有独立转向执行器的全向轮腿移动操作臂，显著提升了在平坦结构化环境中的机动性和操作灵活性。
  2. 算法框架创新：提出了一个接触感知的全身动态优化控制框架，创新性地统一了机械臂的点接触模型和轮子的简化线接触模型，并通过热启动策略实现了高维系统的实时控制。
  3. 建模创新：开发了适用于4WIS-4WID平台的统一运动学模型，消除了不同运动模式间的控制策略切换，提升了控制的一致性和鲁棒性。

局限性：
论文提到，所采用的轮地线接触模型是一种简化的退化模型，释放了滚动方向的力矩约束。这虽然有利于计算，但可能无法完全捕捉高速或大载荷下复杂的轮胎力学特性。此外，框架主要针对低中速（<2 m/s）场景，更高动态下的控制性能有待验证。

对后续研究的启示：

  1. 可以探索更精细的轮地接触模型（如基于滑移率的模型）与优化框架的结合，以进一步提升高速动态性能。
  2. 所提的统一运动学与接触感知优化框架，为其他复合移动操作机器人（如轮足、履带式）的控制提供了可借鉴的思路。
  3. 未来工作可以考虑将学习的方法（如RL）与本文的基于模型的优化框架相结合，利用前者处理不确定性和复杂接触，后者提供精确的模型先验和稳定性保证。

💡 一句话总结

本文针对全向轮腿式移动机械臂，解决其因自由度冗余、复杂轮地接触动力学及运动与操作需协调而导致的统一全身运动控制难题。提出了**接触感知全身动态优化框架**，关键点在于整合了机械臂操作的**点接触模型**与轮地交互的**线接触模型**，并采用**预热启动策略**加速在线优化。同时，为机器人4WIS-4WID驱动方案定制了**统一运动学模型**，避免了不同运动策略间的模式切换。仿真与实验验证了该框架能实现**敏捷地形穿越、高速全向移动与精确操作**，展现了在半结构化环境中应用的潜力。
