Self-Correcting VLA: Online Action Refinement via Sparse World Imagination

arXiv: 2602.21633 作者: Heng Tao Shen Team 日期: 2026-02-25

📝 详细解读

研究背景与动机

标准视觉-语言-动作（VLA）模型依赖于拟合统计数据先验，限制了其对底层物理动态的鲁棒理解。强化学习通过探索增强了物理基础，但通常依赖于与智能体内部状态隔离的外部奖励信号。世界动作模型作为一种新兴范式，整合了想象与控制以实现预测性规划，但其依赖于隐式的上下文建模，缺乏明确的自我改进机制。现有基于强化学习的VLA方法，无论是利用视觉语言模型合成奖励、基于规则设计奖励，还是基于相似性计算奖励，其奖励信号均外在于策略的内部表征，无法实现与智能体内部状态对齐的自我改进。本文针对VLA模型缺乏基于内部状态的、高效的自我改进能力这一痛点，提出了一种新视角：通过稀疏的未来状态想象，内在地指导动作精炼，从而实现自我改进。本文的核心思路是提出一个名为SC-VLA的两阶段框架，首先通过稀疏世界想象（SPI）预测任务进度和未来轨迹趋势，约束策略编码短期物理演化；然后通过在线动作精炼（OAR）模块，利用预测的未来状态构建稠密的内在奖励，对基础动作进行残差调整，实现策略的在线自我校正。

方法详解

SC-VLA是一个两阶段框架，集成了稀疏世界想象（SPI）与基于残差强化学习的在线动作精炼（OAR）。其奖励信号是内生的，完全源自世界想象内部的一致性，而非外部的稠密监督。

图1：SC-VLA方法整体框架。该框架通过稀疏世界想象（SPI）预测未来状态，并利用在线动作精炼（OAR）进行基于残差强化学习的策略优化，实现自我改进。

整体流程：第一阶段，模型接收多视角图像和语言指令，通过VLM骨干网络提取中间层特征作为条件信号，并输入由当前状态、任务进度查询、状态变化查询和动作查询构成的增强查询序列。一个基于DiT的骨干网络同时生成基础动作，并从中间层特征解码出预测的任务进度和未来相对状态变化。第二阶段，将预测的稀疏世界状态（当前状态、预测进度、预测状态变化）作为观测，输入一个轻量级的残差策略网络，该网络输出一个动作残差。最终执行的动作是冻结的基础策略动作与可学习的残差策略动作的加权和。整个精炼过程由基于预测状态变化构建的内生稠密奖励指导。

核心模块一：稀疏世界想象 (SPI)
该模块旨在为策略注入短期物理演化先验。首先，使用SigLIP-2编码多视角图像，与语言指令拼接后，由Eagle-2 VLM处理，并提取其中间层特征 o_mid 作为条件信号。其次，构建包含物理预测目标的增强查询序列 q_input = [s_t, q_pt, q_Δst, q_a]，其中 s_t 为当前本体状态，q_a 为动作轨迹查询，q_pt 和 q_Δst 分别用于预测任务进度 p_t 和未来相对状态变化 Δs_t。Δs_t 定义为未来某时刻相对于当前时刻的末端执行器位置、旋转（欧拉角）和夹爪开合的相对变化，在局部坐标系下计算以增强泛化性。最后，在DiT骨干网络中，从第 m 个中间块提取隐藏特征 h^(m)，通过两个独立的MLP头预测进度 p_t^ 和状态变化 Δs_t^。总训练损失为流匹配损失与两个辅助预测损失的加权和：L_total = L_FM + λ1 L_prog + λ2 L_Δst。这使得模型在生成动作的同时，内部化了一个连贯且可解释的稀疏世界表征。

图2：SC-VLA架构图。上方第一阶段利用VLM和DiT骨干生成基础动作和稀疏世界想象；下方第二阶段实施在线动作精炼，残差RL模块通过学习残差项来优化最终动作，该过程由源自想象一致性的内生稠密奖励指导。

核心模块二：在线动作精炼 (OAR)
该模块旨在不从头学习的情况下，使策略适应分布外扰动和细粒度接触。它采用残差策略结构，其输入观测为稀疏世界想象 o_w = (s_t, p_t^, Δs_t^)。残差策略 π_res 被建模为一个轻量MLP参数化的高斯策略，并使用SAC算法进行优化。最终动作 a_t = a_t_base + λ a_t_res，其中 a_t_base 来自冻结的基础策略。为了在稀疏环境奖励下提供连续指导，基于预测的短期状态变化（仅位置部分）构建方向性稠密引导奖励：r_t_guide = ( (P_t+n - P_t) · (P_goal - P_t) ) / ( ||P_t+n - P_t|| ||P_goal - P_t|| + ε )，其中 P_goal = P_t + Δs_t^pos。该奖励衡量了实际位移与预测演化方向的对齐程度。此外，引入动态权重调度，使用预测进度 p_t^ 作为调度信号，使预测指导在任务早期占主导，在后期逐渐减弱。最终奖励为：r_t_final = η(p_t^) · w_guide · r_t_guide + r_t_env - c。

创新点：1) 内生自我改进：摒弃对外部奖励模型、手工规则或轨迹相似度的依赖，通过策略内部预测的稀疏未来状态构建奖励，实现与内部状态对齐的自我优化。2) 稀疏物理想象：将高维的未来状态预测简化为低维、可解释的任务进度和相对状态变化预测，作为物理演化的明确约束。3) 残差精炼与动态调度：在冻结的强基础策略上叠加轻量残差调整，并结合基于进度的动态奖励调度，平衡了探索效率与精细操作阶段的自主性。

实验与结果

实验设置：在仿真环境ManiSkill3（基于SAPIEN）中评估，选取了四个具有挑战性的操作任务：StackCube、PlaceSphere、LiftPegUpright和PegInsertion。每个任务使用100条演示进行训练，评估50个回合。对比基线包括：基于扩散的Diffusion Policy (DP)、基于动作分块的ACT、基于流匹配的π0，以及作为基础模型的GR00T N1.5。主要评估指标为任务成功率和成功回合的平均完成步数（用于衡量吞吐量）。

定量结果：如表1所示，SC-VLA（仅SPI）在平均成功率上已超越所有基线。引入在线动作精炼（OAR）后，SC-VLA (SPI, OAR) 取得了最佳性能，平均成功率达到86%。在最难的PegInsertion任务上，相比π0和GR00T N1.5分别提升了34%和16%。从OAR带来的提升（Δ from OAR）看，在多个任务上均有显著增益。

图3：各方法在ManiSkill任务上的成功率与平均完成步数对比。SC-VLA在成功率上达到最优，同时保持了最短的平均步数，体现了高成功率和高效性。

如表2所示，SC-VLA (SPI, OAR) 取得了最短的平均完成步数（157步），相比π0减少了43%，相比DP减少了8%，证明了其更高的执行效率和任务吞吐量。OAR模块在除LiftPegUpright外的所有任务上都显著减少了执行步数，尤其在PegInsertion任务上减少了34%。

消融实验：论文对关键组件进行了系统消融。结果显示，移除稀疏世界想象（w/o SPI）或移除在线动作精炼（w/o OAR）都会导致性能下降，证明了两个模块的必要性。使用固定的、非预测性的未来目标（w/o Δs）或使用静态权重而非动态调度（w/o Scheduling）也会损害性能，验证了基于预测的奖励和动态调度的有效性。

图4：在StackCube任务上的消融研究结果。移除非预测性目标（w/o Δs）或动态调度（w/o Scheduling）均导致性能下降，验证了这些组件的贡献。

真实世界实验：在ARX5机器人平台上进行的真实世界实验表明，SC-VLA相比基础模型GR00T N1.5，成功率提升了14%，并且对初始位置扰动和不同物体材质展现出更强的鲁棒性。

图5：真实世界实验的成功率对比。SC-VLA相比基础模型GR00T N1.5取得了显著提升。

图6：在存在位置扰动和不同物体材质情况下的成功率。SC-VLA表现出更强的鲁棒性。

组件贡献总结：1) 稀疏世界想象（SPI）：通过预测任务进度和未来状态变化，为策略提供了物理演化先验，显著提升了基础策略在复杂接触任务上的精度和成功率。2) 在线动作精炼（OAR）：通过残差调整和内生稠密奖励，使策略能在线适应扰动，进一步提升了成功率和执行效率，尤其在精细操作任务上效果显著。3) 动态权重调度：确保了策略从早期的预测指导平滑过渡到后期的自主探索，是维持策略在任务全程稳定收敛的关键。

总结与启发

核心贡献：1) 提出了SC-VLA框架，首次将稀疏世界想象与在线残差精炼相结合，实现了VLA模型完全基于内部预测的内生自我改进。2) 设计了稀疏世界想象模块，通过预测低维、可解释的任务进度和状态变化，为策略编码物理演化提供了明确约束。3) 开发了在线动作精炼模块，利用预测状态构建内生稠密奖励，并引入动态调度，实现了在保持基础策略稳定性的同时进行高效精细调整。

局限性：论文指出，当前方法依赖于基础策略提供的合理初始动作和预测，若基础策略在特定场景下完全失败，残差精炼的改进空间可能有限。此外，稀疏想象目前主要关注末端执行器的短期状态，未来可探索更丰富的物理属性预测。

后续研究启示：1) 内生奖励设计：本研究展示了不依赖外部模型或复杂计算，仅通过策略内部预测构建有效奖励的可行性，为强化学习中的奖励设计提供了新思路。2) 分层策略改进：残差精炼范式为大规模预训练模型的高效、轻量级在线适应提供了可扩展的框架。3) 物理理解与控制的统一：将可解释的物理状态预测作为策略的中间监督信号，是弥合感知、理解与控制之间鸿沟的有力途径。

💡 一句话总结

这篇论文提出Self-Correcting VLA (SC-VLA)模型，旨在解决现有视觉-语言-动作模型依赖静态数据先验、对物理动态理解不足的核心问题。其关键技术包括：1）**稀疏世界想象**，通过预测任务进度和未来轨迹趋势来编码短期物理演化；2）**在线动作精炼**，基于预测的未来状态重塑奖励，动态优化策略。实验表明，该方法在机器人操作任务上实现了最先进的性能，相比基线方法减少了16%的步骤，成功率提高了9%，并在真实世界实验中获得了14%的性能提升。
