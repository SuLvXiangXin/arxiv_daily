Mass-Adaptive Admittance Control for Robotic Manipulators

arXiv: 2504.16224 作者: Gholampour, Hossein, Slightam, Jonathon E., Beaver, Logan E. 日期: 2025/04/22

📝 详细解读

研究背景与动机

在工业、医疗等领域，机器人需要在动态环境中实现精确运动和柔顺的力调节。导纳控制因其能使机器人根据外部力调整运动而被广泛应用，但其在处理未知或变化的负载质量时面临挑战。传统方法难以实时区分并适应由未知负载引起的力，这常导致末端执行器显著下垂、轨迹跟踪误差增大，甚至任务失败。尽管存在基于传感器、自适应阻尼或模糊逻辑等方法，但在柔顺操作中实时处理未知负载仍存在局限。

本文针对机器人操作中未知负载质量导致末端执行器下垂、破坏轨迹跟踪精度这一具体痛点，提出了将实时质量估计集成到自适应导纳控制器中的新视角。核心思路是：利用力/力矩传感器数据实时估计未知负载质量，并动态生成一个虚拟激励力来补偿该负载重力，从而在保持系统柔顺性的同时，显著减少轨迹跟踪误差。

方法详解

本文提出的质量自适应导纳控制方法，其整体框架是在经典导纳控制回路中，增加了一个实时质量估计与补偿模块。导纳控制器根据末端力传感器测量的外部力，计算期望的末端速度指令，并发送给机器人内置的运动控制器执行。质量自适应模块则在机器人运动过程中，利用传感器数据估计负载质量，并生成一个反向的激励力注入导纳控制律，以抵消负载重力影响。

图2：导纳控制架构框图。控制器根据测量的外部力 F_ext，计算参考速度 \dot{p} 发送给机器人。机器人与环境交互形成闭环。

核心模块一是基础导纳控制器。其设计目标是实现具有虚拟质量 M_a、阻尼 B_a 和刚度 K_a 的二阶动力学系统，关系式为：F_ext(t) = M_a(p¨_a - p¨_0) + B_a(p˙_a - p˙_0) + K_a(p_a - p_0)。通过求解此方程得到导纳加速度 p¨_a，积分后得到速度指令 p˙_a 发送给机器人。

核心模块二是质量自适应模块，用于解决未知负载 m_u 引起的下垂问题。如图3所示，下垂位移 δ_z = mg / K_zz。为补偿此影响，在导纳动力学方程中引入激励力 F_exc，修改后的公式为：F_ext = M_a(p¨_a - p¨_0) + B_a(p˙_a - p˙_0) + K_a(p_a - p_0) - F_exc。

图3：未知负载质量 m_u 导致末端执行器从期望位姿 p_0 下垂 δ_z。

质量估计的关键技术细节基于牛顿第二定律。已知夹爪质量 m_g，通过将FT传感器测得的力 f(t) 和末端加速度 p¨(t) 投影到垂直方向 z^，可实时估计未知质量：m_u = (f_z(t)) / (p¨_z(t)) - m_g，其中 f_z 和 p¨_z 为垂直方向分量。随后，利用估计的质量计算激励力：F_exc = m_u * p¨_z * z^，并将其注入控制律。

本文的创新点具体体现在：1) 提出了一个统一的、将实时在线质量估计与导纳控制相结合的自适应框架；2) 通过推导出的激励力直接补偿未知负载的动力学效应，而非单纯调整控制器参数；3) 进行了稳定性分析，指出在自由空间中，只要多项式 (M_a s^2 + B_a s + K_a)(M_u - M^_u(s))s + R(s) = 0 的根具有负实部，质量估计器就是稳定的。

实验与结果

实验平台为UR5e六自由度机械臂，配备腕部FT传感器、内部加速度计和Robotiq 2F-140夹爪。控制算法在Linux PC上运行，通过RTDE与机器人通信。实验评估了一个具有挑战性的“箱到架”拾取放置任务，任务涉及六个路径点，其中路径点3用于抓取物体，路径点6用于放置。

图4：X-Z平面上的六个规划路径点示意图。展示了从抓取到放置的完整运动轨迹。

实验对比了不同虚拟刚度 K 设置下，启用与禁用质量补偿的性能。所有试验均搬运1500g的负载。关键参数包括：虚拟质量 m 设为4kg，阻尼 b 根据临界阻尼公式 b = 2√(m·k) 设置，刚度 k 在100-2500 N/m间变化，位置误差阈值 ϵ 为3.5mm。

图5：实验轨迹对比图。展示了不同实验条件下机器人的实际运动路径与参考路径的偏差情况。

关键实验结果总结如下：

  1. 实验1（中刚度，无补偿）：任务失败，下垂8.1mm。
  2. 实验2（高刚度，无补偿）：任务成功，下垂3.5mm（达到阈值）。高刚度能抑制下垂但降低了系统柔顺性。
  3. 实验3（低刚度，有补偿）：任务成功，下垂**<3.5mm**。这是最重要的结果，表明在低刚度（高柔顺性）设置下，质量补偿能有效将下垂减少到阈值以下，使任务得以完成。
  4. 实验4（低刚度，无补偿）：任务失败，下垂高达46.8mm。这凸显了在低刚度下，未知负载导致的严重下垂问题，以及质量补偿的必要性。

图6：各实验Z方向位置误差对比图。清晰显示了启用质量补偿（Exp 3）在低刚度下能将误差维持在极低水平，而未补偿时（Exp 4）误差巨大。

消融实验的核心在于对比“启用/禁用质量补偿”以及“不同刚度水平”。结果表明，质量补偿组件是在保持低刚度（高柔顺性）的同时实现高轨迹精度的关键。没有它，要保证精度就必须使用高刚度，从而牺牲了柔顺性。

总结与启发

本文的核心贡献在于：1) 提出了一个将实时质量估计与导纳控制相统一的自适应框架，能同时保证柔顺性与轨迹精度；2) 推导了基于传感器数据的在线质量估计算法及对应的激励力补偿项；3) 通过实验验证了该方法能使机器人在低刚度（高柔顺）设置下，可靠地完成带有未知负载的复杂操作任务，将下垂误差从46.8mm降低到3.5mm以下。

论文自身提到的局限性包括：稳定性分析主要针对自由空间工况；质量估计模型假设负载为纯质量，未考虑复杂的动力学效应。

这项工作对后续研究的启示包括：可将此自适应框架扩展到包含接触力的更复杂任务中；探索处理非刚性或动态变化负载的方法；研究将更先进的质量与惯性参数在线估计方法（如递归最小二乘法）集成到该控制框架中，以进一步提升性能。

💡 一句话总结

本文针对机器人操作中负载质量未知或变化导致控制误差与不稳定的核心问题，提出了一种质量自适应导纳控制方法。该方法将导纳控制框架与在线质量估计器相结合，通过动态更新激励力来实时补偿未知负载，从而抑制末端执行器下垂并保持系统稳定性。实验在带有横杆的货架拾放任务中验证了该方法的有效性，与基线导纳控制方案相比，显著提升了路径点跟踪精度与柔顺运动性能。
