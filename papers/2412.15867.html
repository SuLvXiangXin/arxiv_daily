IRGS: Inter-Reflective Gaussian Splatting with 2D Gaussian Ray Tracing

arXiv: 2412.15867 作者: Gu, Chun, Wei, Xiaofei, Zeng, Zixuan, Yao, Yuxuan, Zhang, Li 日期: 2024/12/20

📝 详细解读

研究背景与动机

当前，基于3D高斯泼溅（3DGS）的逆向渲染方法成为主流，它们通过为每个高斯基元赋予材质属性来实现高效的场景重建与分解。然而，由于3DGS依赖光栅化进行渲染，其无法准确模拟逆向渲染所必需的、基于光线追踪的效应，例如入射光的可见性（visibility）和间接辐射（indirect radiance）计算。这一局限性导致现有方法（如GS-IR）不得不采用简化版的渲染方程（如split-sum近似），或像R3DG那样使用可学习参数来近似间接光照，从而无法准确建模真实的互反射（inter-reflection）效应，限制了材质与光照估计的准确性。本文针对缺乏高效高斯光线追踪器这一具体痛点，提出了一种新的2D高斯光线追踪技术，并将其集成到高斯泼溅框架中，旨在不简化渲染方程的前提下，准确计算入射光并捕获复杂的互反射效果。其核心思路是：提出可微分的2D高斯光线追踪技术，以在预训练的2D高斯泼溅模型上直接进行光线追踪，从而动态、准确地计算渲染方程所需的可见性与间接辐射。

方法详解

IRGS的整体框架分为两个阶段。第一阶段，预训练一个标准的2D高斯泼溅模型，以获得可靠的场景几何（深度、法线）。第二阶段，基于此几何进行逆向渲染，优化材质（漫反射率albedo、粗糙度roughness）和环境光照。其核心创新在于，在评估渲染方程时，使用提出的2D高斯光线追踪技术来动态查询入射光的直接辐射（来自环境贴图）和间接辐射（来自场景互反射）。

图2：IRGS框架示意图。从一组带有材质属性的2D高斯出发，通过光栅化生成albedo、roughness、位置和法线图。然后在对应位置进行分层采样以评估渲染方程。入射光的辐射度被分解为来自环境贴图的直接辐射L_dir，以及通过2D高斯光线追踪获得的、带有可见性V的间接辐射L_ind。

核心模块1：2D高斯光线追踪
这是实现准确互反射计算的关键。与在3D高斯上进行光线追踪（3DGRT）不同，2D高斯圆盘具有明确定义的射线-泼溅交点。给定一个由中心点μ、不透明度o、两个主切向量t_u/t_v和缩放向量s定义的2D高斯，该方法首先为每个2D高斯构建一个自适应二十面体网格作为其边界代理，确保包含最小影响因子α_min。这些三角形被用于构建BVH，并利用OptiX进行硬件加速的射线-三角形求交。通过k-buffer每射线排序算法，获得每条射线相交的高斯序列的精确顺序。射线与2D高斯圆盘的交点p通过解析计算获得（公式6），进而计算该高斯在交点处的影响值（公式7）。最终，射线的RGB值和累积不透明度通过从前到后alpha混合相交的2D高斯得到（公式8）。此过程完全可微，支持通过反向传播优化间接光照。

图3：在预训练的高斯泼溅模型上直接应用高斯光线追踪的性能对比。与3DGRT相比，本文提出的2D高斯光线追踪（2DGRT）在定量指标和视觉结果上的质量下降都显著更小。

核心模块2：逆向渲染流程与优化
在第二阶段，对于每个像素，根据第一阶段得到的深度图确定表面交点位置x。在该点，使用完整的渲染方程（公式3）计算出射辐射度Lo。入射辐射度Li(ωi, x)被分解为直接部分L_dir（来自可优化环境贴图）和间接部分L_ind。间接辐射的计算需要从交点x向入射方向ωi发射次级射线，并通过2D高斯光线追踪查询击中点的辐射度。可见性V同样通过向光源方向发射射线并检查是否被遮挡（即光线追踪的累积不透明度ort > 阈值）来确定。BRDF采用简化的Disney模型，包含漫反射项fd和镜面反射项fs（公式4）。为了处理渲染方程积分所需的密集蒙特卡洛采样带来的巨大计算开销，论文提出了一种高效的优化方案：在每次迭代中，仅对像素的一个子集（论文中为1/4）完整地评估渲染方程（包括光线追踪），而对于其他像素，则使用一个轻量级的MLP来预测其出射辐射度，该MLP以位置、法线、视线方向等为输入，并通过与完整计算像素的损失进行蒸馏学习。

核心模块3：重光照策略
在场景优化完成后进行重光照时，先前方法常常忽略或近似间接辐射分量。IRGS提出了一种新颖的策略来查询间接辐射：在重光照时，对于给定的新光照条件，通过2D高斯光线追踪实时计算间接光照贡献，从而生成包含真实互反射效果的图像。

创新点总结

  1. 2D高斯光线追踪：实现了在预训练的2D高斯泼溅模型上进行直接、高质量的光线追踪，解决了3D高斯光线追踪存在的交点不一致和质量下降问题。
  2. 完整渲染方程集成：首次在高斯泼溅框架中集成了未经简化的完整渲染方程，通过可微的光线追踪准确计算可见性和间接辐射。
  3. 高效优化方案：通过像素子采样与MLP蒸馏，大幅降低了完整计算渲染方程的计算成本。
  4. 可查询间接辐射的重光照：提供了在任意新光照下计算间接辐射的有效策略。

实验与结果

实验设置：方法在多个标准逆向渲染基准上进行了验证，包括合成数据集（如Glossy Synthetic）和真实世界数据集（如NeRF Object Dataset）。对比的基线方法包括基于NeRF的方法（如NeRFactor、PhySG、InvRender、NeRD）和基于3DGS的方法（如R3DG、GS-IR、GaussianShader）。评估指标包括新视角合成的图像质量指标（PSNR、SSIM、LPIPS）以及材质估计的准确性（albedo的MSE、粗糙度的MSE）。

关键实验结果：

  • 新视图合成与重光照：在合成和真实数据集上，IRGS在新视图合成和重光照任务上均取得了最佳或极具竞争力的性能。例如，在Glossy Synthetic数据集上，其PSNR（27.21）显著高于R3DG（25.69）和GS-IR（25.68）。重光照结果的视觉对比显示，IRGS能准确再现复杂的互反射效果（如颜色渗出），而其他方法则无法建模或产生错误。
  • 材质分解：在材质估计任务上，IRGS也展现出优越性。例如，在估计albedo的MSE指标上，IRGS（0.008）低于R3DG（0.013）和GS-IR（0.012），表明其分解出的材质更准确。

图4：与基线方法在重光照任务上的定性对比。IRGS能够准确重建复杂的互反射效果（如红色球在绿色平面上的颜色渗出），而其他方法（R3DG, GS-IR）则无法捕捉这些效果或产生不准确的光照。

图5：消融实验。从左至右分别为：(a) 真实图像，(b) 不使用间接光照（仅直接光）的结果，(c) 不使用所提高效优化策略（全像素采样）的结果，(d) IRGS完整模型结果。实验表明，间接光照对建模互反射至关重要，而高效优化策略在保持视觉质量的同时大幅提升了训练速度。

消融实验总结：
论文进行了消融实验以验证各个组件的贡献：

  1. 间接光照（Indirect Lighting）：移除间接光照（即仅使用直接光照）会导致互反射效果完全缺失，验证了其必要性。
  2. 高效优化方案（Efficient Optimization）：移除该方案（即对所有像素进行完整渲染方程计算）虽然能产生可比的结果，但训练时间大幅增加（约4倍），证明了该方案在效率和性能间的良好平衡。
  3. 2D高斯光线追踪 vs 3D高斯光线追踪：使用3DGRT替代2DGRT会导致渲染质量显著下降（PSNR从27.21降至25.87），并引入视觉伪影，证明了2D高斯光线追踪的优越性。
  4. 训练损失函数：消融了第一阶段训练中的各项几何约束损失（法线一致性损失、深度失真损失、边缘感知平滑损失、掩码损失），证明它们共同作用以获得更清晰、更准确的几何，这对后续的逆向渲染至关重要。

总结与启发

核心贡献：

  1. 提出了2D高斯光线追踪技术，实现了在预训练2D高斯泼溅模型上高效、高质量的可微分光线追踪。
  2. 提出了互反射高斯泼溅框架，首次将完整渲染方程无缝集成到高斯泼溅中，通过2D高斯光线追踪动态计算入射光，从而准确建模复杂的互反射效应。
  3. 设计了一种高效的优化方案，以可承受的计算成本处理渲染方程评估所需的密集采样。
  4. 提出了一种新颖的重光照策略，能够在优化后的场景中查询间接辐射。

局限性：
论文提到，尽管采用了高效的优化策略，由于需要执行光线追踪，其方法在训练速度上仍然慢于纯粹基于光栅化的方法。此外，该方法依赖于第一阶段预训练得到的高质量几何。

启示：
IRGS为高斯泼溅在逆向渲染乃至更广泛的基于物理的渲染任务中的应用开辟了新途径。它证明了将精确的光线追踪能力与高效的高斯表示相结合是可行的，并且对提升材质与光照估计的物理准确性至关重要。后续研究可以围绕进一步加速光线追踪过程、扩展至更复杂的材质模型（如各向异性、金属材质）或动态场景，以及探索在资源受限环境下的应用展开。

💡 一句话总结

本文提出IRGS方法，解决基于3D高斯泼溅的逆渲染中因缺乏高效光线追踪而难以准确建模入射光可见性与间接辐射度的问题。核心是引入可微分的2D高斯光线追踪技术，动态计算入射辐射度，并应用完整渲染方程而不简化；同时设计高效优化方案以处理蒙特卡洛采样计算需求。在多个标准基准测试上的实验验证了IRGS能有效捕捉复杂互反射效果，实现更准确的材料和光照估计。
