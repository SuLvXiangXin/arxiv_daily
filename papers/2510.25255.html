Time-Optimal Transport of Loosely Placed Liquid Filled Cups along Prescribed Paths

研究背景与动机

在工业机器人领域，时间最优的运动规划是核心关注点。对于某些应用，机器人末端需要沿着一条预先定义的几何路径运动，这被称为时间最优路径跟随问题。然而，优化过程不仅需要考虑机器人自身的动力学限制，还必须处理与具体任务相关的约束。本文聚焦于一类复杂的任务约束——“服务员运动问题”，即使用机器人末端执行器上的托盘来运输一个松散放置的、装满液体的杯子。现有方法在处理此类带有内部动态（液体晃动）的物体运输时面临巨大挑战，因为剧烈的加速度会导致液体溢出。本文针对这一痛点，提出将液体晃动动力学整合到最优控制问题的动态模型中，从而在规划时间最优轨迹时主动抑制晃动、防止溢出。核心思路是：在满足机器人动力学、物体稳定性（防抬起、滑动、倾倒）以及液体晃动约束的前提下，求解使运输时间最短的轨迹。

方法详解

本文方法的核心是将时间最优路径跟随问题扩展为一个包含复杂任务约束的最优控制问题（OCP），并采用直接多重打靶法求解。

整体框架与问题建模：系统状态定义为 $x = (\sigma, \dot{\sigma}, \ddot{\sigma}, x_R^T, x_F^T)^T$，其中 $\sigma$ 为路径参数，$x_R$ 为机器人旋转关节的状态（位置、速度、加速度），$x_F$ 为液体摆模型的状态（角度、角速度）。控制输入为 $u = (\sigma^{(3)}, q_R^{(3)})$，即路径参数和旋转关节的加加速度（Jerk）。优化目标是最小化加权后的终端时间 $t_E$ 和控制输入能量。整个最优控制问题的约束包括：机器人关节位置、速度、加速度的上下限；确保杯子稳定在托盘上的力约束（非抬起、非滑动、非倾倒）；以及限制液体晃动的状态约束。

核心模块与技术细节：

  1. 时间最优路径跟随基础：给定末端耦合点H的参数化路径 $^I\mathbf{r}_H(\sigma)$，目标是为路径参数$\sigma(t)$找到时间最优演化，同时满足机器人运动学约束。
  2. 机器人动力学模型：采用形式为 $M(q)\ddot{q} + G(q, \dot{q})\dot{q} + g(q) = Q_M$ 的动力学方程。由于前三轴为线性轴且旋转轴臂长很短，文中未直接限制电机扭矩，而是限制了加速度。
  3. 松散放置物体的任务约束：将杯子建模为刚体，通过分析其与托盘接触点的约束力/力矩（接触力偶 $Q_{obj}^z = (f^{zT}, M^{zT})^T$）来制定稳定性条件：
    • 非抬起条件：法向接触力 $f_z \ge 0$。
    • 非滑动条件：切向摩擦力 $\sqrt{f_x^2 + f_y^2} \le \mu_0 f_z$，其中 $\mu_0$ 为静摩擦系数。
    • 非倾倒条件：倾倒扭矩 $\sqrt{M_x^2 + M_y^2} \le r_o f_z$，其中 $r_o$ 为杯底半径。
  4. 液体填充杯子的任务约束：将液体简化为一个附着在杯子上的球形摆模型（长度L，质量m，粘性阻尼d）来近似其晃动动力学。

    图2：液体填充杯子的球摆模型示意图（y-z视图）。液体被建模为悬挂在点F_P的摆锤，其摆动角度 $\phi, \theta$ 受到约束以防止溢出。
    摆的运动方程推导为 $\ddot{q}_F = f_F(q, \dot{q}, \ddot{q}, q_F, \dot{q}_F)$，其中 $q_F = (\phi, \theta)^T$。为防止液体溢出，对摆的角度和角速度施加约束：$x_F \le x_F \le \bar{x}_F$。

创新点：与仅考虑路径几何或刚体运输的方法相比，本文的主要创新在于将液体晃动动力学作为状态变量和状态约束直接纳入最优控制问题的框架中，从而在轨迹规划阶段就主动抑制晃动，而非仅通过末端运动条件或反馈控制来应对。该方法统一处理了机器人动力学、物体接触力学和液体内部动力学之间的耦合。

总结与启发

核心贡献：

  1. 提出了一套系统性的方法，将液体晃动动力学整合到时间最优路径跟随的最优控制问题中，为“服务员运动问题”提供了一个完整的建模与求解框架。
  2. 明确了运输松散放置液体杯子所需的多重物理约束（机器人运动学/动力学、物体接触稳定性、液体内部动力学），并给出了其数学表述。
  3. 通过数值仿真验证了采用直接多重打靶法能够有效求解这一高度非线性、带复杂约束的最优控制问题，生成满足所有约束的时间最优轨迹。

局限性：论文自身指出，所使用的液体球摆模型是对复杂流体动力学的一种简化。未来的工作需要考虑液体质心移动对约束力计算的影响，以提升模型精度。此外，目前结果仅为仿真验证，需要在真实机器人上进行实验测试。

后续研究启示：本文展示了将复杂任务的具体物理约束（尤其是包含内部动态的对象）纳入轨迹优化层面的可行性。该方法可扩展至其他涉及柔性、流体或颗粒状物料运输的机器人操作任务。同时，如何将简化模型在真实世界中产生的误差通过在线估计或鲁棒控制进行补偿，是值得探索的方向。

💡 一句话总结

本文解决机器人沿预定路径运输松散放置液体填充杯的时间最优问题，核心在于最小化液体晃动以避免溢出。关键技术包括将晃动动力学建模为球形摆并纳入最优控制问题，采用直接多重打靶法在CasADi框架中求解。通过约束稳定杯体位置，防止滑动和倾斜，实现了时间最优路径跟随，有效解决了服务员运动问题。
