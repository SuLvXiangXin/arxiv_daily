Optimal Robotic Velcro Peeling with Force Feedback

arXiv: 2506.05812 作者: Volkan Isler Team 日期: 2025-06-06

📝 详细解读

研究背景与动机

在非结构化环境中，机器人需要操作各类物体，包括如带有魔术贴的鞋子等变形体。魔术贴剥离任务（例如协助穿脱鞋、打开袋子）集中体现了在感知能力有限、环境未知情况下的机器人操作核心挑战。当前，涉及接触的操作通常依赖视觉或腕部力传感器，但力反馈仅能提供接触区域的局部信息，且难以关联到非接触区域的状态，在涉及变形体时模糊性更高。作者先前的工作表明，基于视觉的强化学习（RL）方法能提升性能，但在仅依赖力反馈的“最小感知”设定下，无模型RL方法与最优策略间仍存在显著性能差距。此外，未知的表面几何形状为状态估计带来了不确定性。

本文针对的具体痛点是：在表面几何形状任意且未知的情况下，机器人仅依靠末端执行器的位置和力反馈来剥离魔术贴。这带来了部分可观测性挑战（魔术贴的铰链状态无法直接观测）以及传感器反馈不完整的问题。本文提出了一种新视角：首先通过一个独立的控制器确保剥离部分始终处于拉紧状态，从而避免处理复杂的“触觉虚空”问题；在此基础上，基于准静态动力学假设建立简洁的解析模型来描述系统。核心思路是：在拉紧假设下，推导全观测情况下的闭式最优解作为基准，然后针对部分可观测情况，设计一个结合了贝叶斯状态估计（粒子滤波）和启发式控制的方法，在状态估计精度和任务效率之间进行权衡，以实现高效、鲁棒的剥离。

方法详解

本文方法的整体框架是一个基于准静态模型的状态估计与控制循环。系统状态通过粒子滤波器进行估计，估计的状态用于启发式控制器决策，控制器产生的动作执行后，新的力与位置观测又用于更新状态估计。

核心模块1：状态与动作建模
首先引入关键假设：剥离的魔术贴部分始终保持拉紧。在此假设下，任意时刻的剥离状态可由铰链点（Hinge）和末端执行器（Tip）描述。

图2：魔术贴剥离状态示意图。状态 $X = [h_x, h_y, \theta, \phi, r]$，其中 $(h_x, h_y)$ 为铰链点位置，$\theta$ 为附着部分在铰链处的方向角，$\phi$ 为剥离部分的方向角，$r$ 为剥离部分的长度。

动作分为两类：剥离动作与非剥离动作。

图3：魔术贴剥离动作。(a) 剥离动作：末端从 $E_t$ 移动到 $E_t+1$，参数为方向 $\alpha$ 和距离 $d$，会导致铰链点移动。(b) 非剥离动作：仅改变剥离部分方向角 $\phi$（变化量 $\delta \phi$），铰链点和剥离长度不变。动作参数化为 $A = [\alpha, d, s, \delta \phi]$，$s$ 为动作类别指示符（0-剥离，1-非剥离）。

基于准静态假设，可以推导剥离动作下的状态转移几何约束。

图4：剥离动作下的状态转移。满足几何约束：$|E_t H_t| + |H_t H_{t+1}| = |E_{t+1} H_{t+1}|$。通过方程组(3)可由当前状态 $X_t$ 和动作 $A_t$ 求解下一状态 $X_{t+1}$ 中的 $h_x, h_y, r, \phi$。值得注意的是，$\theta_{t+1}$ 在该约束中未出现，表明附着部分的方向 $\theta$ 是一个与历史状态和动作无关的隐藏参数，这构成了状态估计的主要挑战之一。

观测模型为 $O = [t_x, t_y, \beta]$，包括末端位置 $(t_x, t_y)$ 和力传感器测得的拉力方向角 $\beta$。

核心模块2：基于粒子滤波的状态估计器
为解决部分可观测性和 $\theta$ 的不可观测性，本文设计了基于粒子滤波的状态估计器 $\mathscr{F}$，并采用了状态空间分解策略。滤波器根据动作类别和观测类型，选择性地更新状态参数的子集，以避免干扰未观测参数的分布。定义了三个测量模型用于更新粒子权重：

  1. 力方向模型（式5）：$\beta = \phi + \pi + \mathcal{N}(0, \sigma_1^2)$，用于更新 $\phi$。
  2. 末端位置模型（式6）：$[t_x, t_y]^T = [h_x, h_y]^T + r * [\cos \phi, \sin \phi]^T + \mathcal{N}(0, \sigma_2^2)$，用于更新 $h_x, h_y, r, \phi$。
  3. $\theta$ 更新模型（式7）：$\theta_{t+1} = \theta_t + \mathcal{N}(0, \sigma_3^2)$，用于更新 $\theta$。由于真实表面（如圆弧、拐角）不总符合零均值高斯模型，本文还设计了一个辅助估计模型，结合基于主成分分析（PCA）的直线拟合和基于最小二乘的圆弧拟合算法，对历史估计的铰链点位置进行拟合，以先降低 $\theta_t$ 的偏差，再应用高斯模型。

更新策略如下：

  • 执行剥离动作时：状态参数耦合，为避免增加粒子偏差和方差，仅使用模型1（$\mathscr{F}_1$）更新 $\phi$。
  • 执行非剥离动作时：$h_x, h_y, r$ 与 $\phi$ 解耦，使用模型1和2（$\mathscr{F}_1, \mathscr{F}_2$）更新粒子。
  • $\theta$ 的更新：可由控制器决定何时触发，使用模型3（$\mathscr{F}_3$）进行更新。

核心模块3：启发式控制器
控制器基于当前估计的状态，通过启发式规则在“探索”（改善状态估计）和“利用”（高效剥离）之间进行权衡。其核心逻辑是维持一个“健康指数” $z$，该指数反映了状态估计的不确定性。算法流程如下：

  1. 对齐阶段：如果当前估计的 $\phi - \theta \neq \pi/2$（即剥离方向并非垂直于附着方向，这是高效剥离的条件），则执行一个非剥离动作将 $\phi$ 调整至 $\pi/2 + \theta$，然后执行一个短距离的试探性剥离动作，并更新状态估计（$\mathscr{F}_1, \mathscr{F}_2$）。
  2. 决策阶段：计算当前健康指数 $z$。以概率 $z$ 决定进入“探索”模式（$s=1$），否则进入“利用”模式（$s=0$）。
  3. 探索模式：执行一个非剥离动作（改变 $\phi$）以收集信息，更新状态估计（$\mathscr{F}_1, \mathscr{F}_2$），然后可能触发 $\theta$ 的更新（$\mathscr{F}_3$）。
  4. 利用模式：执行一个沿当前最优方向（$\theta + \pi/4$）的剥离动作，并更新状态估计（$\mathscr{F}_1$）。

创新点总结：

  1. 提出了保持剥离部分拉紧的控制器，简化了系统动力学。
  2. 基于准静态假设建立了可处理的解析状态、动作和转移模型。
  3. 设计了基于状态空间分解的粒子滤波器，能有效处理隐藏参数 $\theta$ 和传感器噪声。
  4. 提出了一个显式权衡探索与利用的启发式控制器，其决策基于状态估计的不确定性（健康指数）。

实验与结果

实验设置：在仿真环境中进行验证，设置了三种具有复杂几何不确定性的表面轮廓：1) 平面、2) 圆弧、3) 拐角。实验考虑了真实的传感器噪声。

对比基线：

  1. 随机策略：随机选择动作。
  2. 无模型强化学习（RL）方法：基于作者先前工作[1]的PPO算法。
  3. 本文方法。
  4. 全观测最优解（Oracle）：作为理论性能上限。

关键实验结果：
本文方法在三种几何形状上均达到了 100% 的成功率。在能量成本方面，相比全观测最优解（Oracle），本文方法的成本增长**小于80%**，显著优于随机策略和RL基线。

图5：平面轮廓下的性能对比。本文方法（红色）成功率100%，能量成本接近Oracle（绿色），远低于RL（蓝色）和随机策略（紫色）。

图6：圆弧轮廓下的性能对比。本文方法依然保持100%成功率和较低的能量成本增长，而RL和随机策略性能下降明显。

图7：拐角轮廓下的性能对比。在最具挑战性的拐角场景中，本文方法仍能维持100%成功率，能量成本控制良好，显著优于基线方法。

消融实验：

图8：消融实验结果。(a) 移除启发式控制器中的探索部分（“No Exp.”），导致在拐角处因$\theta$估计错误而失败。(b) 移除$\theta$的辅助拟合模型（“No Aux.”），在圆弧和拐角场景下$\theta$估计误差增大，性能下降。(c) 在状态估计中忽略动作类别，对所有参数进行全更新（“No Dec.”），引入了额外偏差和方差，降低了成功率并增加了能量消耗。

消融实验总结：

  • 探索机制：对于估计隐藏参数 $\theta$ 至关重要，移除后无法处理几何突变。
  • $\theta$ 辅助拟合模型：显著提高了在非平坦表面（圆弧、拐角）上 $\theta$ 估计的准确性。
  • 状态空间分解更新策略：有效防止了不恰当的更新引入估计偏差，提升了滤波器的整体鲁棒性。所有组件都对最终性能有重要贡献。

总结与启发

核心贡献：

  1. 针对涉及频繁强力交互的变形体操作任务，提出了一种新颖的解决方案，推进了该领域的研究。
  2. 提出了一个完整的框架，结合了解析建模（准静态动力学）、贝叶斯状态估计（基于状态空间分解的粒子滤波）和启发式控制（探索-利用权衡），以解决力反馈下的魔术贴剥离问题。
  3. 在包含复杂几何不确定性和真实传感器噪声的仿真环境中，验证了方法的有效性与鲁棒性，实现了100%成功率，且能量成本仅比全观测最优解增加不到80%，大幅超越基线方法。

局限性：

  1. 方法依赖于“剥离部分始终拉紧”的假设，这需要一个额外的控制器来维持，且在处理极度松弛或纠缠的魔术贴时可能不适用。
  2. 实验仅在仿真中进行，未在真实机器人平台上进行验证。

对后续研究的启示：

  1. 本文的框架（建模-估计-控制）可扩展至其他仅依靠接触传感的机器人操作任务，如操作门、抽屉或其它可变形线性物体。
  2. 启发式控制器可以进一步优化，例如通过更精细的信息论指标（如信息增益）来指导探索，或与模型预测控制（MPC）结合。
  3. 可以探索如何将少量、可能不完整的视觉信息（如初始抓取点）融入状态估计框架，以进一步提升初始状态估计的准确性，从而加速任务完成。

💡 一句话总结

本文研究机器人在表面几何形状任意且未知时，仅依靠力反馈和末端位置反馈剥离魔术贴的核心问题。针对环境部分可观测的挑战，提出基于准静态动力学假设的建模方法，设计状态估计器从力/位置反馈中估计状态，并开发启发式控制器平衡探索与利用行为。实验表明，在复杂几何不确定性和传感器噪声环境下，该方法成功率达100%，能量成本较完全可观测最优解仅增加不到80%，显著优于基线方法。
