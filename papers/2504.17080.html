Geometric Formulation of Unified Force-Impedance Control on SE(3) for Robotic Manipulators

arXiv: 2504.17080 作者: Seo, Joohwan, Prakash, Nikhil Potu Surya, Lee, Soomi, Kruthiventy, Arvind, Teng, Megan, Choi, Jongeun, Horowitz, Roberto 日期: 2025/04/23

📝 详细解读

研究背景与动机

在涉及与环境交互的机器人操作任务中，阻抗控制是主要的控制方案之一，常与操作空间公式结合使用，以确保与未知环境的安全交互。近年来，基于学习的策略（如模仿学习）在从视觉输入生成期望的末端执行器轨迹方面取得了成功。然而，对于高精度、接触丰富的任务，仅输出期望位姿的策略可能不足。因此，近期工作也提出了输出阻抗/导纳增益或直接力曲线的方法。但改变增益或施加直接力会引入稳定性挑战，因为原始静态增益阻抗/导纳控制器的稳定性在增益变化时无法保证。机器人-环境交互的稳定性分析通常与无源性概念紧密相关，如果机器人的闭环系统被设计为无源的，则无源性定理可以提供整个交互动力学的稳定性保证。

统一力-阻抗控制（UFIC）被提出作为一种手段，使机器人操作器能够在执行阻抗控制任务时保持与环境的接触并施加期望的力。UFIC使用能量储罐增强来确保闭环系统的无源性。然而，UFIC存在两个主要局限性：首先，它没有考虑操作器末端执行器位姿描述中固有的SE(3)流形结构，而是简单地将位姿误差视为笛卡尔向量进行处理；其次，为了利用储罐增强建立阻抗控制项的无源性，必须定义一个修改后的期望速度，然后对其进行积分以生成新轨迹，这使下一步期望速度的更新复杂化，并可能导致控制过程中的因果性崩溃。

本文针对上述痛点，提出了在SE(3)流形上对统一力-阻抗控制进行几何化表述的新视角。其核心思路是：在先前提出的几何阻抗控制（GIC）框架基础上，通过引入能量储罐增强和速度/力场，构建一个几何统一力-阻抗控制（GUFIC），在保证系统无源性和力跟踪性能的同时，充分尊重SE(3)的流形结构，并解决了原UFIC中的因果性实现问题。

方法详解

本文提出的几何统一力-阻抗控制（GUFIC）框架旨在将力跟踪控制无缝集成到几何阻抗控制中，同时通过能量储罐保证整个闭环系统的无源性。整体控制律为 F' = F'_f + F'_i，其中 F'_f 是修改后的力跟踪控制项，F'_i 是修改后的阻抗控制项，最终关节扭矩为 T = J_b^T F'。

核心模块包括三个部分：1) 基于能量储罐的力跟踪控制器；2) 基于能量储罐和速度场的阻抗控制器；3) 速度场与力场生成器。

1. 力跟踪控制器的储罐增强：原始的力跟踪控制律 F_f 是一个PID控制器，用于跟踪期望力场 F_d 与传感器测量外力 -F_e 之间的误差。为保证该控制端口 (V^b, F_f) 的无源性，引入能量储罐 T_f = 0.5 * x_{tf}^2。储罐状态 x_{tf} 的动态根据 (V^b)^T F_f 的符号（由指标 γ_f 判断）和储罐能量水平（由指标 β_f 和 α_f 调节）进行更新。最终，力跟踪控制律被修改为 F'_f = (γ_f + α_f (1-γ_f)) F_f，该修改确保了力跟踪端口不会向系统注入净能量。

2. 阻抗控制器的储罐增强与速度场：几何阻抗控制律 F_i 本身能保证无源性，但其期望速度 V^*_d 和期望位姿 g_d 是预先给定的时间轨迹。当与力跟踪项 F'_f 和外力 F_e 耦合时，需要对端口 (V^*_d, -(F'_f+F_e)) 也进行无源性管理。为此引入第二个能量储罐 T_i = 0.5 * x_{ti}^2。其动态更新类似于力跟踪储罐，但依赖于 (V^*_d)^T (F'_f+F_e) 的符号（γ_i）和能量水平（β_i, α_i）。关键的一步是，根据储罐状态修改期望速度：(V^*_d)' = (γ_i + α_i (1-γ_i)) V^*_d，并由此定义新的速度跟踪误差 e'_V = V^b - (V^*_d)' 和通过积分得到的新期望轨迹 g'_d。

然而，直接修改和积分 (V^*_d)' 会导致因果性问题：当前时间步修改速度后积分得到新轨迹，但下一时间步的期望速度无法因果性地从这个新轨迹导出。为解决此问题，本文创新性地引入了速度场概念。速度场 V^*_d(t, g) 是一个将当前时间 t 和当前位姿 g 映射到 se(3) 元素的函数，定义为：V^*_d(t,g) = Ad_{g_{ed}} \bar{V}^b_d(t) + ζ ∇_1 Ψ(g, \bar{g}_d)。其中第一项是将原始期望速度 \bar{V}^b_d(t) 平移到当前位姿，第二项是一个基于位姿误差度量 Ψ 的梯度项，用于将系统拉向原始期望轨迹 \bar{g}_d(t)。这样，期望速度不再仅是时间的函数，而是位姿和时间的函数。修改 (V^*_d)' 现在意味着修改这个场函数，而新的期望位姿 g'_d 可以通过积分 g'_d 的微分方程 \dot{g}'_d = g'_d (V^b_d)' 得到，整个过程是因果的。

3. 力场：为了与速度场表述一致，本文也将期望力表述为力场 F_d(t, g)，即它可以依赖于时间和当前位姿。在本文的实验中，为简化起见，使用了在空间和时间上均为常数的力场。

4. 最终控制律与无源性证明：结合以上模块，修改后的阻抗控制律为：
F'_i = \tilde{M} (\dot{V}^*_d)' + \tilde{C} (V^*_d)' + \tilde{G} - f_G(g, g'_d) - K_d e'_V
总控制律为 F' = F'_f + F'_i。论文的核心定理（定理1）证明了，在假设1（雅可比满秩、轨迹可微等）下，采用GUFIC控制律的机器人闭环系统对于端口 (V^b, F_e) 是无源的，其存储函数为 S_tot = S + T_i + T_f，其中 S 是位姿误差的动能与势能之和。

与现有方法的创新点：

  1. 流形结构整合：与UFIC将平移和旋转误差分开处理不同，GUFIC在SE(3)李群结构内一致地处理所有位姿误差。
  2. 因果性实现：通过引入速度场，解决了UFIC中因修改期望速度而导致的非因果实现难题。
  3. 等变性优势：由于构建于GIC框架之上，GUFIC控制律具有SE(3)等变性，这在将学习算法融入控制律时，能显著提高样本效率和学习迁移性。

实验与结果

实验设置：研究在Mujoco仿真环境中进行，使用了Neuromeka的Indy7机器人模型。控制循环和仿真的采样频率为1000 Hz。力/扭矩传感器读数通过一个截止频率为5 Hz的二阶低通滤波器。

对比方法：主要的对比基线是标准的几何阻抗控制（GIC）。在GIC中，为了对表面施加期望的力，需要表面的精确几何信息或自适应律。在实验中，假设GIC控制器只有粗略的猜测信息，因此会导致不正确的力施加。

实验场景：

  1. 场景1：圆形轨迹：机器人末端执行器在保持特定姿态的同时，跟踪一个空间中的圆形轨迹，并对表面施加法向力。期望力场为 F_d = [0, 0, 10, 0, 0, 0]^T N（仅z方向力）。
  2. 场景2：球面线轨迹：机器人末端执行器在球面上跟踪一条轨迹（包含位置和姿态变化），同时对球面施加法向力。这是一个典型的SE(3)运动任务。

关键实验结果：

  • 轨迹跟踪性能：在两个场景中，GUFIC和GIC都展示了近乎完美的轨迹跟踪性能。图2和图5分别展示了两个场景下位置分量的跟踪情况，两者均能紧密跟随期望轨迹。

图2：场景1的轨迹跟踪结果。GUFIC和GIC在x、y、z方向均能很好地跟踪期望的圆形轨迹。

图5：场景2的轨迹跟踪结果。GUFIC和GIC在x和y方向完美跟踪，在z方向由于球面信息不完善存在微小稳态误差。

  • 力跟踪性能：这是GUFIC的核心优势。如图3和图7所示，在两种场景下，GUFIC都能成功收敛到期望的10N法向力。而仅使用GIC时，由于缺乏精确的模型信息来调整阻抗参数，产生了显著的稳态误差，无法实现准确的力控制。

图3：场景1的力跟踪结果。GUFIC能准确跟踪10N的期望力，而GIC由于模型信息不准确，存在较大的稳态误差。

图7：场景2的力跟踪结果。GUFIC成功收敛到期望力，而GIC则存在显著误差。

  • 能量储罐动态：图4和图8展示了两个场景中力控制储罐 T_f 和阻抗控制储罐 T_i 的能量变化。在整个任务执行过程中，储罐能量始终保持在设定的上下限之间，未被耗尽，这表明无源性条件始终得到满足，任务得以成功执行。

图4：场景1中GUFIC的能量储罐 T_f 和 T_i 变化。储罐能量在安全范围内波动，未被耗尽。

图8：场景2中GUFIC的能量储罐 T_f 和 T_i 变化。阻抗控制储罐 T_i 初始值较高，在任务初期快速下降，但随后保持稳定，系统保持无源。

  • 误差收敛：图6展示了场景2中基于距离度量 Ψ 的位姿误差。该误差逐渐收敛，证明了系统在位置和姿态动力学上的跟踪误差均趋于零。

图6：场景2的位姿误差函数 Ψ。误差逐渐收敛，表明在平移和旋转动力学上的跟踪误差均趋近于零。

消融实验总结：论文虽然没有进行严格的组件消融实验，但通过将GUFIC与基础的GIC进行对比，清晰地展示了将力跟踪与基于储罐的无源性管理、速度场相结合的整体框架的有效性。对比实验结果表明，缺少这些组件的GIC无法在模型信息不完善时实现精确的力跟踪。

总结与启发

核心贡献：

  1. 几何化统一：首次在SE(3)流形上完整表述了统一力-阻抗控制，将平移和旋转误差在李群结构内进行一致性处理，继承了几何阻抗控制的数学严谨性和优雅性。
  2. 解决因果性难题：通过引入速度场（和力场）的概念，创新性地解决了原UFIC框架中修改期望速度导致的非因果实现问题，使控制器更易于实现。
  3. 保证无源性并赋能学习：通过双能量储罐机制，保证了机器人与环境交互时的全局无源性，从而确保了接触稳定性。同时，控制器的SE(3)等变性为后续集成学习算法提供了样本效率和学习迁移性方面的潜在优势。

局限性：

  • 论文提到了在接触丢失（力与运动控制平行）的情况下，原UFIC框架可能存在稳定性问题。本文虽采用了控制器整形函数 ρ 来缓解，但未对其性能进行深入分析。
  • 实验中使用的力场是空间和时间上的常数，未充分展示力场 F_d(t,g) 更一般化应用的性能。
  • 目前工作仅在仿真环境中进行了验证，尚未在真实机器人平台上进行实验。

对后续研究的启示：

  1. 学习与控制的结合：GUFIC的SE(3)等变性和无源性特性，使其成为一个理想的底层控制器，可与上层学习策略（如学习期望轨迹、力场或阻抗参数）安全、高效地结合。
  2. 复杂接触任务：该框架非常适合需要同时进行精细运动控制和力调节的复杂接触式操作任务，如装配、打磨、精密插拔等。
  3. 动态力场与自适应：未来工作可以探索如何在线学习或自适应地生成时变、位姿相关的力场 F_d(t,g)，以应对更复杂多变的环境。

💡 一句话总结

本文针对机器人操作中需在未知环境下实现高精度力跟踪并保证系统被动性的核心问题，提出了几何统一力阻抗控制（GUFIC）框架。该方法基于统一力-阻抗控制（UFIC）和几何阻抗控制（GIC），在SE(3)流形上采用微分几何公式化，利用能量储罐增强同时实现力跟踪与阻抗控制以确保被动性，并通过引入速度与力场解决了UFIC的非因果实现问题。GUFIC继承了SE(3)不变性与等变性，有助于提升学习控制中的样本效率。在模拟环境中验证了该控制律能有效跟踪包含位置和方向的SE(3)轨迹，并实现在表面上的力施加。
