Overcoming Dynamic Environments: A Hybrid Approach to Motion Planning for Manipulators

arXiv: 2504.06596 作者: Ngo, Ho Minh Quang, Nguyen, Dac Dang Khoa, Le, Dinh Tung, Paul, Gavin 日期: 2025/04/09

📝 详细解读

研究背景与动机

在动态和不确定环境中运行的机械臂需要高效的运动规划，以在避开障碍物的同时保持平滑的轨迹。当前主流方法包括采样运动规划器（SBMPs）如RRT*，以及人工势场（APF）及其变体速度势函数（VPF）。SBMPs能生成渐近最优路径，但在动态环境中需要频繁重规划，计算成本高昂。VPF等反应式规划器计算效率高、实时性好，但存在局部最小值、振荡行为、目标收敛困难以及复杂约束下性能不佳等问题。这两种方法都需要大量的环境先验知识，这在不确定环境中是一个挑战。

本文针对动态未知环境中，单一规划范式难以兼顾全局最优性、实时适应性和计算效率的痛点，提出了一个混合规划的新视角：将SBMP的全局路径规划能力与VPF的实时反应能力相结合。本文的核心思路是：利用SBMP一次性生成关节空间的全局最优路径作为“向导”，然后设计一个结合动态前瞻路径跟踪与VPF局部反应的控制框架，使机械臂能在跟踪全局路径的同时，实时、低计算开销地避开动态障碍物。

方法详解

本文提出的混合规划方法分为两个主要阶段：离线/初始化阶段的全局路径规划，以及在线执行阶段的混合路径跟踪与局部避障。

图4：混合运动规划框架概览。左侧（蓝色）为离线阶段：在关节空间使用RRT*进行全局路径规划，并对路径进行平滑（B样条）和时间参数化。右侧（橙色）为在线阶段：结合了基于动态前瞻的全局路径跟踪控制器和基于VPF的局部反应式控制器，共同生成关节速度指令。

1. 关节空间全局路径规划
该方法首先在机械臂的关节空间（而非任务空间）使用SBMP（如RRT*或RRT-Connect）规划一条从起点到终点的路径。选择关节空间是为了避免任务空间到配置空间的复杂映射，并便于在关节配置之间进行平滑插值。原始SBMP路径通常较为粗糙，因此使用B样条曲线对路径点进行拟合以获得平滑的几何路径，随后对该路径进行时间参数化，生成满足运动约束的轨迹。该全局路径可表示为 N 步的关节配置序列：𝐐 = {𝐪₁, 𝐪₂...𝐪_N}，其中 𝐪 是包含 n 个关节角的向量。

图2：7自由度机械臂在杂乱环境中关节空间的RRT*搜索。绿色点表示末端执行器沿全局路径（以固定时间步长）的位置，展示了运动的平滑性。

2. 在线混合控制框架
在线阶段，控制器融合了全局路径跟踪和局部避障反应。其输出是关节速度指令 𝐪˙，计算公式为：𝐪˙ = 𝐪˙_track + 𝐪˙_react。其中，𝐪˙_track 来自全局路径跟踪控制器，𝐪˙_react 来自局部VPF反应式控制器。

  • 全局路径跟踪控制器（动态前瞻）：该控制器的作用不是几何上精确跟踪已被障碍物阻挡的全局路径，而是将其作为引导机械臂走向目标的“向导”。它采用了一种改进的纯追踪算法，其核心是动态前瞻点选择机制。前瞻点 𝐪next 的选择不仅基于当前配置在路径上的最近点 𝐪_x，还综合考虑了当前关节速度的幅值 ‖𝐪˙‖ 和局部路径曲率 κ。具体通过算法1实现：前瞻步长 s = int(k_v‖𝐪˙‖ + k_cκ + s_base)，其中 k_v 和 k_c 为权重系数。高速或路径平直时前瞻更远以平滑运动，低速或高曲率时前瞻更近以保证精度。最终 𝐪_next = 𝐪{x+s}。跟踪控制律则基于PD控制：𝐪˙_track = 𝐊_P(𝐪_next − 𝐪(t)) + 𝐊_D 𝐪˙_e。

  • 局部反应式控制器（改进的VPF）：该控制器在任务空间（末端执行器）定义，用于实时避开动态障碍物。它由吸引速度场和排斥速度场组成。

    • 吸引速度场：吸引末端执行器朝向“下一个目标姿态”（由全局路径上的前瞻点通过正运动学计算得出）。其本质是一个比例控制器：𝐯_att = -K_att 𝐞，其中 𝐞 是当前姿态与目标姿态之间的空间误差（包含平移和旋转）。
    • 排斥速度场：驱使机械臂远离障碍物。论文重点改进了对移动障碍物的处理。如图3所示，排斥速度 𝐯_rep 的大小不仅取决于与障碍物的距离 d，还考虑了障碍物相对速度 𝐯_obs 的方向（夹角 ψ）。当障碍物迎面而来（ψ 小）时，排斥力更强；当障碍物同向或背离运动时，排斥力减弱。排斥速度方向沿障碍物到机械臂最近点的连线方向。最终，任务空间速度 𝐯_ts = 𝐯_att + 𝐯_rep 通过机械臂的雅可比矩阵伪逆映射到关节空间：𝐪˙_react = 𝐉⁺ 𝐯_ts。

图3：机械臂单个连杆与移动障碍物的场景。向量 d 为连杆到障碍物最近点的连线，ψ 为 d 与障碍物速度 v_obs 的夹角，用于计算排斥速度。

创新点：1) 混合架构：SBMP一次性全局规划与VPF持续局部反应的结合，在保证全局导向性的同时极大降低了动态环境中的重规划计算负担。2) 动态前瞻路径跟踪：在关节空间引入基于速度和曲率的动态前瞻机制，使路径跟踪更适应机器状态和路径形状。3) VPF的改进：在排斥势场中加入了障碍物相对运动方向的因素，使避障反应更符合直觉和高效。

实验与结果

实验设置：方法在7自由度机械臂模型上进行了验证。仿真实验使用V-REP/CoppeliaSim平台。实物实验使用了Sawyer机器人，并配备OptiTrack运动捕捉系统来跟踪动态障碍物。

对比方法：本文方法与以下基线进行了对比：1) RRT*：作为纯SBMP代表。2) APF：经典人工势场法。3) VPF：纯速度势函数法。4) RRT with Replanning：动态环境中周期性调用RRT重规划。

图5：动态障碍物场景下的轨迹对比。纯VPF（左）和纯APF（中）均陷入局部最小值而失败。本文的混合方法（右）成功利用全局路径引导并结合局部避障到达目标。

关键实验结果：

  1. 成功率：在包含静态和动态障碍物的复杂场景中，纯APF和VPF的成功率较低（分别因振荡和局部最小值失败），RRT*重规划成功率高但计算慢。本文混合方法达到了 100%的成功率。
  2. 路径长度与平滑性：混合方法产生的路径长度与最优的RRT相当，且由于VPF的实时调整和动态前瞻，路径比频繁重规划的RRT更平滑。
  3. 计算效率：这是核心优势。混合方法的在线计算时间远低于需要频繁重规划的RRT。因为其只在初始化时进行一次全局规划，在线阶段仅进行轻量的控制器计算。论文指出，在动态环境中，混合方法的计算开销比实时重规划RRT低一个数量级。
  4. 实物验证：在Sawyer机器人上的实验表明，该方法能有效处理真实世界中的动态障碍物（如移动的盒子），机械臂运动平滑，成功抵达目标。

消融实验：
论文通过实验验证了各核心组件的贡献：

  • 动态前瞻的作用：与固定前瞻步长相比，动态前瞻在路径曲率高的区域能提供更精确的跟踪，在平直区域能实现更平滑、更快的运动。
  • VPF中速度方向因子的作用：考虑障碍物运动方向（ψ角）的排斥场，比仅基于距离的排斥场能产生更合理、更高效的避障动作，减少了不必要的偏离。

总结与启发

核心贡献：

  1. 提出了一种新颖的混合运动规划框架，将SBMP的全局最优性、离线规划特性与VPF的实时性、低计算开销相结合，有效解决了动态未知环境中的机械臂规划问题。
  2. 设计了基于动态前瞻的关节空间路径跟踪控制器，通过结合当前速度和路径曲率自适应选择前瞻点，实现了对全局路径的智能、平滑跟踪。
  3. 对传统的VPF进行了改进，引入了障碍物相对运动方向作为排斥力大小的调节因子，使局部避障行为更加合理和高效。

局限性：论文自身提到，该方法假设环境整体结构相对稳定，仅障碍物动态移动。在环境发生剧烈、全局性变化（如场景布局根本改变）时，初始的全局路径可能完全失效，此时需要重新进行全局规划。此外，方法对于高度动态、密集的障碍物环境的极限性能仍有待进一步测试。

对后续研究的启示：

  1. 混合范式：证明了结合不同规划范式（全局/局部，最优/反应式）是解决复杂动态环境问题的有效途径，可启发更多样的组合方式（如优化局部规划器）。
  2. 人机协作：论文指出未来可应用于人机协作场景，该框架能够以低延迟响应人类的突然运动，同时保持整体的任务导向性，对于实现安全、高效的共享 workspace 具有潜力。
  3. 自适应重规划触发机制：可以进一步研究如何智能地判断何时初始全局路径已“过时”，从而自动触发轻量级的局部重规划或全局路径更新，以应对更剧烈的环境变化。

💡 一句话总结

本文针对机械臂在动态不确定环境中运动规划的挑战，提出一种混合方法。核心问题是传统速度势场（VPF）规划器易陷于局部最优且不稳定，而基于采样的运动规划器（SBMP）重新规划计算成本高。解决方案是将改进的VPF与SBMP相结合：SBMP负责生成全局最优路径，VPF提供对动态障碍物的实时适应与避障。该方法在真实机械臂上通过动作捕捉系统验证，结果表明其提升了运动性能并降低了计算开销。
