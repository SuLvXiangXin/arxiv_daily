Robot Learning Using Multi-Coordinate Elastic Maps

arXiv: 2505.06092 作者: Hertel, Brendan, Azadeh, Reza 日期: 2025/05/09

📝 详细解读

研究背景与动机

机器人学习操作技能的一种便捷方法是通过示教学习（LfD）。当前，许多LfD方法致力于在特定坐标空间中编码技能特征。例如，多坐标成本平衡方法（MCCB）将演示编码到多个微分坐标（如笛卡尔、切向、拉普拉斯）中，并通过优化确定各坐标的重要性以生成复现轨迹。然而，MCCB使用高斯混合模型（GMM）进行编码，在面对噪声演示时可能产生不够平滑的复现轨迹。

另一方面，弹性地图（Elastic Maps）作为一种灵活的统计方法，因其内置的平滑性约束和计算效率，在LfD中显示出优势。但传统的弹性地图主要关注笛卡尔空间中的近似。本文针对现有方法（如MCCB）在应对噪声演示时平滑性不足，以及传统弹性地图未能充分利用多坐标空间特征的问题，提出了一种新视角：将多坐标表征的思想与弹性地图的框架相结合。

本文的核心思路是：提出多坐标弹性地图（MC-Elmap），在弹性地图的优化框架中同时引入笛卡尔、切向和拉普拉斯坐标空间的近似能量项，并自动优化各坐标空间的权重，从而在保留技能多维度特征（如位置、速度轮廓、形状）的同时，确保生成轨迹的平滑性。

方法详解

MC-Elmap 的整体流程是：输入一个或多个时间对齐的演示轨迹，将其分别转换到笛卡尔、切向和拉普拉斯坐标空间；通过期望最大化（EM）算法迭代优化：在E步根据当前复现轨迹节点对演示数据进行聚类，在M步基于新的聚类结果和自动调优的权重，求解一个凸优化问题以更新复现轨迹节点；最终输出优化后的轨迹。

该方法的核心模块是经过修改的弹性地图能量函数。传统弹性地图包含三种能量：近似能量 $u_{\mathcal{X}}$（使复现轨迹逼近演示数据）、拉伸能量 $u_E$（使节点间距均匀）、弯曲能量 $u_R$（使节点排列平直）。MC-Elmap 的创新在于引入了切向坐标和拉普拉斯坐标的近似能量项 $u_{\mathcal{T}}$ 和 $u_{\mathcal{L}}$。

具体而言，给定 $N$ 条演示轨迹 $\mathcal{D}$，首先通过图切向矩阵 $\mathbf{T}$ 和图拉普拉斯矩阵 $\mathbf{L}$ 将其转换到切向空间（$\bm{D}{\mathcal{T},i} = \mathbf{T}\bm{D}i$）和拉普拉斯空间（$\bm{D}{\mathcal{L},i} = \mathbf{L}\bm{D}i$）。将所有演示数据在三个空间中分别拼接，得到 $\bm{g}, \bm{g}{\mathcal{T}}, \bm{g}{\mathcal{L}}$。复现轨迹由 $M$ 个节点 $\bm{y}$ 表示。通过聚类（每个数据点归属于欧氏距离最近的节点）确定聚类矩阵 $\bm{K}$ 和权重矩阵 $\bm{W}$。

新的总能量函数为：
$$f_0(y) = u_{\mathcal{X}} + u_{\mathcal{T}} + u_{\mathcal{L}} + u_E + u_R$$
其中：

  • $u_{\mathcal{X}} = w_{\mathcal{X}}||\bm{W}\bm{y} - \bm{K}\bm{g}||_2^2$
  • $u_{\mathcal{T}} = w_{\mathcal{T}}||\bm{W}\bm{y}{\mathcal{T}} - \bm{K}\bm{g}{\mathcal{T}}||2^2$，其中 $\bm{y}{\mathcal{T}} = \mathbf{T}\bm{y}$
  • $u_{\mathcal{L}} = w_{\mathcal{L}}||\bm{W}\bm{y}{\mathcal{L}} - \bm{K}\bm{g}{\mathcal{L}}||2^2$，其中 $\bm{y}{\mathcal{L}} = \mathbf{L}\bm{y}$
  • $u_E = \lambda||\bm{E}\bm{y}||_2^2$
  • $u_R = \mu||\bm{R}\bm{y}||_2^2$

与现有方法相比，MC-Elmap 的主要创新点在于：1) 在弹性地图中首次整合了多坐标空间（切向、拉普拉斯）的约束；2) 提出了针对多坐标权重 $w_{\mathcal{X}}, w_{\mathcal{T}}, w_{\mathcal{L}}$ 的自动调优方法。自动调优分为两步：首先计算缩放因子 $\beta$ 来平衡不同坐标空间量级的差异，然后通过一个内部优化（最小化所有演示轨迹在笛卡尔空间的重建误差之和）来求解最终的权重 $\alpha$，最终权重 $w = \alpha \beta$。这使得方法对超参数不敏感，易于使用。

图1：MC-Elmap 在真实世界书写任务上的应用。使用UR5e机械臂进行了多次低质量的“R”字形演示（难以在平滑绘制的同时施加足够压力）。然而，MC-Elmap 能够成功且平滑地复现该技能，并强调了演示中未明显呈现的特征。

整个优化问题（公式8）是凸的，可以快速高效求解，并且可以纳入任意数量的起始点、终点或路径点约束。算法通过EM迭代直至聚类不再变化，达到局部最优解。

实验与结果

本文在28个仿真实验和1个真实世界实验中验证了MC-Elmap。使用的数据集/基准包括：LASA手写数据集、自定义的“NShape”轨迹、以及一个真实的机械臂书写任务。实验平台涉及仿真和真实的UR5e机械臂。

对比的基线方法包括：标准弹性地图（Elmap）、多坐标成本平衡（MCCB）、以及动态运动基元（DMP）。评估指标使用了动态时间规整（DTW）误差和任务成功率。

关键实验结果如下：在LASA数据集上，MC-Elmap取得了最低的平均DTW误差（0.012 ± 0.005），优于MCCB（0.014 ± 0.007）和Elmap（0.015 ± 0.007）。在“NShape”轨迹的噪声演示测试中，MC-Elmap的复现轨迹最为平滑且准确。

图2：在LASA数据集上，MC-Elmap（红色）与基线方法复现轨迹的定性对比。MC-Elmap的轨迹在保持形状的同时更为平滑。

图3：不同方法在LASA数据集上的DTW误差分布箱线图。MC-Elmap（中位数为0.011）的整体误差低于MCCB（0.013）和Elmap（0.014）。

图4：“NShape”轨迹的噪声演示（灰色）与MC-Elmap复现结果（红色）对比。MC-Elmap有效滤除了噪声，恢复了平滑的原始形状。

在带有障碍物的“Rail”仿真任务中，MC-Elmap在满足避障约束的前提下，成功率为86%，与MCCB（83%）相当，但显著高于DMP（0%）。在真实的UR5e书写任务中（图1），MC-Elmap成功生成了平滑的“R”字形轨迹。

图5：Rail避障任务示意图。复现轨迹（蓝色）需要在保持演示（灰色）形状的同时，避开障碍物（红色圆柱）。

图6：真实书写任务的详细结果。左侧为嘈杂的原始演示，右侧为MC-Elmap生成的光滑、连贯的复现轨迹。

消融实验总结了各坐标空间的贡献：仅使用笛卡尔坐标（即传统Elmap）在噪声下平滑但可能丢失关键形状特征；引入拉普拉斯坐标对于保留轨迹的几何形状至关重要；而切向坐标有助于保持速度轮廓。MC-Elmap通过自动平衡三者，实现了最佳的综合性能。

总结与启发

本文的核心贡献在于：1) 提出了多坐标弹性地图（MC-Elmap），首次将多微分坐标空间表征集成到弹性地图框架中，实现了对技能多维度特征的编码与平衡复现；2) 引入了针对多坐标权重的自动超参数调优方法，提升了方法的易用性和鲁棒性；3) 通过大量仿真与真实实验验证了MC-Elmap在平滑性、准确性和约束满足方面的优势，特别是在处理噪声演示时。

论文自身提到的局限性包括：目前主要测试了三种预定义的微分坐标（Cartesian, Tangent, Laplacian），未来可以探索其他类型的坐标或自动学习最相关的特征空间；此外，对于在线适应的讨论相对简单。

这项工作对后续研究的启示是：将几何/微分坐标表征与灵活的统计建模（如弹性地图）相结合，是提升LfD方法性能的一个有效途径。自动超参数调优的策略可以减轻使用者的负担。该方法框架具有良好的可扩展性，易于融入其他约束或适应在线学习场景，为机器人技能学习的实际应用提供了新的工具。

💡 一句话总结

本文针对机器人示教学习中单一坐标难以全面捕捉技能特征的问题，提出多坐标弹性地图方法。该方法将人类演示编码至多个微分坐标空间，通过改进的弹性地图进行统计建模并自动优化参数权重，以融合不同坐标的重要特征。实验在仿真和UR5e机械臂真实书写任务中验证，该方法能有效从低质量演示中平滑复现技能，并强调原始数据中未显式存在的特征。
