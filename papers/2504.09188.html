Compliant Explicit Reference Governor for Contact Friendly Robotic Manipulators

arXiv: 2504.09188 作者: Gautam, Yaashia, Nechyporenko, Nataliya, Lin, Chi-Hui, Roncone, Alessandro, Nicotra, Marco M. 日期: 2025/04/12

📝 详细解读

研究背景与动机

机器人领域的主流方法，如传统力/位置控制、阻抗控制，虽然简单但无法内在地处理约束。基于优化的策略（如结合线性互补问题和模型预测控制）或强化学习方法可以处理约束并提供安全保证，但通常计算复杂、资源密集。显式参考调节器（Explicit Reference Governor， ERG）作为一种具有严格安全性和稳定性保证的附加安全滤波器，已在机械臂的实时约束控制（如关节/速度限制、输入饱和和避障）中取得成功。然而，现有的ERG版本将接触视为需要避免的事件，不允许机器人与障碍物发生任何接触，即使这种接触可能改善轨迹结果。这限制了机器人在密集障碍物环境中的操作灵活性和效率。

本文针对机器人需要与环境进行安全接触这一具体痛点，提出了一个新视角：通过调节控制器的参考指令，而非直接改变控制律的柔顺性，来实现安全接触操作。本文的核心思路是提出柔顺显式参考调节器（Compliant ERG， C-ERG），它允许机器人在自由运动时避免接触，或在接触时限制系统的总能量，从而安全运行，并能在两种模式间无缝切换，而无需更改底层控制器或使用专门的求解器。

方法详解

整体框架是一个分层控制系统。高层规划器产生目标关节位置参考r。C-ERG作为一个中间层，接收r和当前系统状态x = [q; q̇]，并输出一个经过调节的、安全的参考信号v(t)给底层预稳定控制器。底层控制器采用关节空间PD控制加重力补偿（公式4），其作用是驱使系统跟踪恒定的参考v。C-ERG通过动态地调节v的导数v̇来确保所有约束得到满足。

图1：ERG架构。应用的参考v是导航场ρ与动态安全裕度Δ乘积的积分。它作为一个中间层，连接高层规划器（提供目标r）和底层预稳定控制器。

核心模块包括导航场（Navigation Field， NF）和动态安全裕度（Dynamic Safety Margin， DSM）。

  1. 导航场（NF）：解决“机器人应采取何种稳态可行路径到达目标”的运动学问题。它由吸引项ρ_att（指向目标r）和排斥项ρ_h（指向硬约束h的内部）组成（公式9-11），生成一个从当前参考v指向安全运动方向的向量场ρ(v， r)。
  2. 动态安全裕度（DSM）：解决“机器人沿其路径运动的速度能多快而不违反约束”的动态问题。它通过预测在恒定参考v下系统的未来轨迹（公式14），并评估约束违反的严重程度来计算。对于硬约束h，其DSM Δ_h定义为未来最坏违反程度的相反数（公式12）。对于允许接触的软约束s(q)，本文的关键创新在于构建了处理“或”逻辑的柔顺动态安全裕度。

与现有ERG相比，C-ERG的创新点具体体现在对软约束s(q)的处理上。传统ERG将s(q) < 0视为硬约束，一旦接触发生（s(q) ≥ 0），DSM Δ_s（公式20，类似于Δ_h）将变为非正，导致v̇ = 0，机器人停止。C-ERG则引入了基于能量的安全条件V(x， v) ≤ E_max（公式7），其中V(x， v)是系统的总能量（动能加虚拟势能，公式5）。安全要求变为逻辑“或”约束：s(q) < 0 或 V(x， v) ≤ E_max（公式17）。这等价于一个组合约束c(x， u， v)（公式19）。

为了处理这个“或”约束，C-ERG定义了能量约束的DSM Δ_E = κ_E (E_max - V(x， v))（公式22）。最终的柔顺DSM Δ(x， v)是Δ_h、Δ_s和Δ_E的min-max组合（公式23）：
Δ(x， v) = min( Δ_h， max(Δ_s， Δ_E) )

  • Δ_h > 0 保证硬约束始终满足。
  • max(Δ_s， Δ_E) > 0 保证“或”约束（要么不接触，要么能量低）满足。
  • 取min则确保同时满足两者。

当处于自由运动（s(q) < 0）时，Δ_s > 0，max(Δ_s， Δ_E) = Δ_s，因此Δ = min(Δ_h， Δ_s)，行为类似于传统ERG，能量不受限制。当发生接触（s(q) ≥ 0）时，Δ_s ≤ 0，此时max(Δ_s， Δ_E) = Δ_E。只要系统总能量V低于阈值E_max，就有Δ_E > 0，从而使Δ可能为正，允许v̇ ≠ 0，机器人可以在接触中继续运动。如果能量V接近E_max，Δ_E将趋近于零，从而自动降低v̇直至停止，防止能量超限。这样就实现了两种模式的无缝、安全过渡。

实验与结果

本文通过数值仿真验证C-ERG，使用的系统复杂度递增：单自由度（1-DOF）系统、双自由度（2-DOF）平面机械臂和三自由度（3-DOF）平面机械臂。实验平台为仿真环境。

对比的基线方法是传统的、将接触视为硬约束的ERG（文中称为“标准ERG”或“非柔顺ERG”）。

图2：单自由度系统对比。左：标准ERG在接触后停止。右：C-ERG在接触后继续向目标运动。

图3：单自由度系统能量变化。C-ERG（右）在接触期间将总能量V(x， v)限制在E_max以下，而标准ERG（左）在接触后能量趋于零（系统停止）。

关键实验结果：对于单自由度系统，标准ERG在末端接触障碍物（s(q) ≥ 0）后，v̇立即变为0，机器人停止（图2左）。而C-ERG在接触后，v继续向目标r演化，驱动机器人在保持接触的情况下向前运动（图2右）。同时，在整个接触过程中，系统的总能量V被严格限制在预设的阈值E_max之下（图3右），保证了接触的安全性。

图4：双自由度机械臂任务场景。机器人需要从左侧初始位置运动到右侧目标位置，中间有障碍物。

图5：双自由度机械臂末端执行器轨迹对比。C-ERG（绿色）允许末端与障碍物上表面安全接触并滑过，从而生成更直接、更短的路径。标准ERG（红色）严格避障，导致更长的绕行路径。

图6：双自由度机械臂关节位置q随时间变化。C-ERG（实线）在接触阶段（约1-4秒）关节持续变化，推动末端滑过障碍物；标准ERG（虚线）在约1.7秒检测到接触后，关节运动停止。

对于更复杂的2-DOF和3-DOF系统，C-ERG同样展现出优势。在2-DOF避障任务中（图4），C-ERG允许机械臂末端与障碍物上表面发生安全接触并滑行通过，产生了更短、更直接的轨迹（图5绿色轨迹）。而标准ERG则规划了一条完全无接触、但更长的绕行路径（图5红色轨迹）。关节位置曲线（图6）清晰显示，C-ERG在接触期间持续运动，而标准ERG则停止。

图8：三自由度机械臂在三种不同E_max能量阈值下的总能量V变化。能量被有效限制在各自阈值之下，更高的E_max允许更快的运动。

对于3-DOF系统，仿真显示了接触力（图7）和系统能量（图8）。接触力在C-ERG的控制下被维持在有限范围内。通过调整能量阈值E_max，可以调节机器人在接触时的“刚度”或允许的运动速度：E_max越大，允许的能量越高，机器人运动越快（但接触力可能更大）；E_max越小，系统在接触时更“柔顺”，运动更缓慢（图8）。

消融实验体现在对不同E_max值的测试上，结果（图8）直接证明了能量限制组件Δ_E的有效性。E_max是C-ERG的关键参数，它明确地量化并控制了接触安全性的一个核心方面——系统能量。

总结与启发

本文的核心贡献在于：1) 提出了柔顺显式参考调节器（C-ERG）框架，通过调节参考指令而非控制器增益，实现了机械臂在自由运动与安全接触操作之间的无缝、实时切换。2) 引入了基于总能量的安全接触条件（V(x， v) ≤ E_max），将“避免接触”的硬约束松弛为一个“或”约束，并通过创新的min-max结构动态安全裕度（公式23）来统一处理，无需复杂的优化求解器。3) 通过从单自由度到三自由度的数值仿真，系统地验证了该方法在保证硬约束、限制接触能量、生成更高效轨迹方面的有效性。

论文自身提到的局限性包括：目前的分析针对完全驱动的机械臂动力学模型；假设接触为点接触，且环境为刚性；方法依赖于准确的动力学模型进行轨迹预测。

本文的启示在于：为需要与环境进行物理交互的机器人提供了一种计算轻量、具有严格安全保证的实时控制新思路。所提出的基于能量的安全准则可以与其他底层控制器（如阻抗控制）结合。未来工作可以探索将C-ERG扩展到欠驱动系统、柔性环境接触以及结合感知不确定性处理。

💡 一句话总结

本文针对机器人操作器在接触环境中安全运行的问题，提出了一种顺应性显式参考调节器（C-ERG）。该方法作为高层规划器与底层控制器之间的中间层，通过限制机械臂接触时的总能量来确保安全，并在无接触时不影响系统性能。C-ERG 实现了自由运动与接触操作间的平滑过渡，并严格保证了操作约束。数值实验验证了该方法在复杂系统中的有效性。
