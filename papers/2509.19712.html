TopoCut: Learning Multi-Step Cutting with Spectral Rewards and Discrete Diffusion Policies

arXiv: 2509.19712 作者: Animesh Garg Team 日期: 2025-09-24

📝 详细解读

研究背景与动机

机器人切割可变形物体在食品加工、医疗手术和制造等领域至关重要，许多现实任务需要多步切割序列以将物体分割成复杂或结构化的形状。尽管在固定轨迹的单材料切割方面已有进展，但针对复杂几何形状的目标条件多步切割仍面临重大挑战。现有方法的关键局限性在于：1) 切割后物体可能无法干净分离，导致结果评估模糊；2) 现有评估指标对姿态变化敏感，通常需要显式对齐；3) 连续切割产生的密集拓扑变化难以从稀疏或噪声观测中感知，阻碍了策略学习的有效性。

本文针对上述痛点，提出了一个统一的框架TopoCut，集成了高保真模拟、鲁棒评估和目标条件策略学习。其核心思路是：通过粒子损伤跟踪机制精确感知拓扑变化，利用基于拉普拉斯-贝尔特拉米特征分析的谱奖励实现姿态不变的切割质量评估，并设计一个结合动力学感知感知模块和粒子级离散扩散策略的学习流程，以实现可扩展和可泛化的多步切割。

方法详解

TopoCut框架旨在使机器人能够通过精确的顺序切割动作，将可变形物体分割成目标指定的形状。其整体流程包含三个核心组件：高保真模拟与拓扑发现、姿态不变谱奖励、以及动力学感知策略学习。

图1：TopoCut框架总览。我们定义了三种代表性切割任务——切片（slice）、切条（stick）、切丁（dice），并通过由谱奖励引导的MPPI和稀疏遥操作收集专家数据。随后，一个动力学感知的感知模块从这些数据中提取粒子级、拓扑感知的嵌入，这些嵌入条件化我们的PDDP策略以生成多步切割动作。

1. 高保真模拟与拓扑发现
模拟环境基于FluidLab的移动最小二乘材料点方法（MLS-MPM）构建，并增加了压缩/拉伸损伤跟踪、具有渐进软化的冯·米塞斯塑性以及表面粘附建模等切割特定功能。物体通常建模为多材料体，结合致密的塑性弹性核心和较软的冯·米塞斯外皮，以同时支持断裂和流动。

拓扑发现的核心是粒子损伤跟踪机制。在模拟中，监控每个粒子的变形梯度F_p，当体积雅可比行列式J_p = det(F_p)满足 J_p ≤ (1-ε_c)^m 或 J_p ≥ (1+ε_s)^m 时，粒子被分类为“损伤”，其中ε_c和ε_s是临界压缩/拉伸阈值，m是材料敏感度参数。对于冯·米塞斯材料，当屈服应力低于临界阈值时也会触发损伤。

图2：(a) 基于粒子的损伤跟踪用于拓扑重建。受损粒子（紫色）标记切割界面，我们通过隐式SDF和Marching Cubes从中恢复物体片段。(b) 动力学感知感知：给定当前拓扑状态topo_t和切割表面动作a_t，预测模块F预测下一个拓扑状态topo_{t+1}。感知编码器Φ随后为下游策略学习生成粒子级嵌入。

检测到受损粒子后，系统开始跟踪刀具轨迹，随后将其“雕刻”进物体的符号距离场（SDF）以强制空间分离。SDF定义为 SDF(x) = min_{p∈P} ||x - x_p|| - r_p，其中r_p是粒子影响半径。使用Marching Cubes提取零等值面，应用拉普拉斯平滑，并将网格分割成连通分量。粒子根据与重建表面的接近度被分配簇标识，从而在整个模拟过程中保持显式、一致的拓扑表示。

2. 姿态不变谱奖励
为了在没有显式对齐的情况下鲁棒地比较切割片段与目标形状，引入了一种基于内蕴形状几何的谱奖励。给定点云X，构建一个k近邻图，计算组合拉普拉斯矩阵L = D - W，并求解特征问题 LΦ = ΛΦ，得到捕获刚性运动不变的内蕴几何的光谱描述符，包括特征值{λ_k}和特征向量{φ_k}。

给定两个形状X和Y，定义谱距离 d_spec(X, Y) = α||Λ_X - Λ_Y||_2^2 + β||Φ_X^T Φ_X - Φ_Y^T Φ_Y||_F^2，其中α, β是权重因子。奖励计算为 R_spec(X, Y) = κ · max(0, C - γ d_spec(X, Y))，其中γ控制奖励衰减，κ是归一化因子，C是预定义常数。对于多片段物体，计算所有片段对的奖励并求和，得到最终评估。

3. 动力学感知策略学习
感知模型训练：将切割下的物体拓扑演化公式化为 F: (topo_t, a_t) ↦ topo_{t+1}。输入是带有簇标签f_i的拓扑点云和带有二进制分割掩码g_j的动作点云。两个点云通过最远点采样下采样，并构建两个图：拓扑图G_topo_t（连接具有相同簇标签的k近邻）和动作图G_a_t（连接具有相同切割掩码的点）。共享的感知编码器Φ独立嵌入这两个图：z_topo_t = Φ(G_topo_t), z_a_t = Φ(G_a_t)。图变换器T融合这些嵌入以预测下一个拓扑状态：G_topo_{t+1} = T(z_topo_t, z_a_t)。训练使用两个监督目标：结合倒角距离（CD）、推土机距离（EMD）和豪斯多夫距离（HD）的几何损失L_pos，以及关于簇分配的匈牙利匹配损失L_topo。总损失为 L = λ_pos L_pos + λ_topo L_topo。

PDDP策略：利用预训练的感知编码器Φ，训练一个目标条件的行为克隆策略。策略输入包括当前拓扑观测topo_t、动作历史a_hist和目标点云g。首先计算点级嵌入：z_t = Φ(X_t, F_t) 和 z_g = Encoder(g)。切割动作被表述为每点二进制标签a_t* ∈ {0,1}^N。受分数熵离散扩散启发，将动作预测建模为条件离散扩散过程：干净标签a_t被逐步加噪为ã_t，策略网络s_θ被训练来预测分数函数 s_θ(ã_t, t, a_hist, z_t, z_g) ≈ ∇_{ã_t} log q_t(a_t | ã_t)。策略通过最小化去噪分数熵（DSE）损失进行优化。去噪过程后，预测的分割标识出可切割区域，然后使用支持向量机算法将预测的切割点拟合成切割平面，重建机器人的刀具姿态a_t ∈ SE(3)。

与现有方法相比，TopoCut的创新点具体体现在：1) 新颖的损伤驱动粒子级拓扑发现机制，实现了对复杂切割过程中拓扑演化的精确、实时跟踪；2) 基于谱几何的、姿态不变的奖励函数，解决了传统几何度量对对齐敏感的问题；3) 将拓扑感知嵌入与条件离散扩散策略（PDDP）相结合，通过预测分割掩码再拟合动作的方式，提升了策略的稳定性和泛化能力。

实验与结果

实验设置：实验在基于FluidLab和MLS-MPM的高保真可变形物体模拟器中进行。物体由两层材料构成：致密的塑性弹性内芯和较软、延展性的冯·米塞斯外皮。定义了三种目标条件切割任务：切片（slice）、切条（stick）、切丁（dice）。评估在405个回合上进行，涵盖9种物体几何（5种训练集内，4种训练集外）和3种目标类型，物体姿态在平移、偏航和尺度上随机化。

基线方法：对比了四种基线：启发式控制器（固定切割轨迹）、DP3（3D扩散模型）、3D Diffuser Actor（标准点云扩散模型）以及人类遥操作。此外，还设置了消融实验（Ours w/o Perception），即不使用预训练感知编码器，仅使用原始点云。

评估指标：1) 归一化谱奖励 R^，通过谱相似性捕获切割质量（归一化到人类遥操作分数）；2) 切割数量 N_C，即正确分割出的目标一致片段的数量。

关键实验结果：

图3：通过多步切割序列的定性奖励评估。仅谱奖励（红色）在正确切割时稳步上升，在错误切割后急剧下降，准确反映了形状保真度且姿态不变。其他指标（CD、EMD、HD）在错误的最终切割后仍持续上升。当纠正最后一切时，谱奖励恢复上升（绿色虚线），证实了其鲁棒性。

图3展示了谱奖励在评估切割序列成功与否时的有效性。在一个四步切割实验中，前三步正确，最后一步故意错误。只有谱奖励能准确反映这一变化，在成功时上升，失败时下降，而传统几何度量（CD、EMD、HD）则持续上升，无法捕捉质量下降。

策略性能对比：如表1所示（数据来自论文正文），在训练集内几何上，PDDP（Ours）在切片和切条任务上取得了最高的归一化谱奖励（0.85和0.71），在切丁任务上（0.57）也表现良好，且切割数量高效。在训练集外几何上，PDDP同样展现了强大的泛化能力，各项指标均优于或与最佳基线持平。相比之下，移除感知模块（Ours w/o Perception）会导致性能下降，尤其是在训练集外几何上，表明预训练的拓扑感知嵌入对泛化至关重要。

消融实验总结：消融实验表明，1) 预训练的动力学感知感知模块是提升策略泛化性能的关键组件，其缺失会导致过拟合和在新形状上的性能退化；2) PDDP通过预测分割掩码再拟合动作的范式，相比直接预测动作的扩散模型基线（如3D Diffuser Actor），具有更好的稳定性和噪声容忍度。

图4：三种规范任务的切割轨迹示例。每一行对应一个任务——切片（上）、切条（中）、切丁（下），展示了初始状态、目标形状以及第1-4步的连续刀具姿态和结果片段。

图4提供了PDDP策略生成的切割序列的定性可视化，展示了其能够针对不同目标（切片、切条、切丁）规划并执行出一系列连贯的切割动作，最终产生与目标形状匹配的片段。

总结与启发

核心贡献：

  1. 高保真模拟与拓扑发现：开发了一个基于粒子弹性塑性求解器的模拟环境，并引入了一种新颖的损伤驱动拓扑发现机制，能够精确跟踪多个切割片段。
  2. 姿态不变谱奖励：提出了一种将拓扑发现与基于拉普拉斯-贝尔特拉米特征分析的谱奖励模型相结合的综合奖励设计，实现了对切割质量一致且鲁棒的评估。
  3. 动力学感知策略学习：提出了一个集成的策略学习流程，其中动力学感知感知模块预测拓扑演化并生成粒子级、拓扑感知的嵌入，以支持PDDP（基于粒子的分数熵离散扩散策略）进行目标条件策略学习。

局限性：

  1. 计算瓶颈：网格重建阶段主导了计算负载，限制了模拟器以实时速度运行，阻碍了高效的大规模在线强化学习。
  2. 任务限制：当前评估仅限于每个切割片段的精确形状匹配，未探索更通用的任务（如仅匹配最大块的雕刻任务）。
  3. 模拟真实性：由于所用MPM算法固有的脆弱性和分辨率限制，平台无法完全支持灵巧的切割控制，模拟高刚度物体（如土豆）等高真实感场景仍具挑战性。

后续研究启示：

  1. 通过算法优化、GPU加速或并行重建管线来缓解网格重建的计算瓶颈，以实现更可扩展的策略训练。
  2. 扩展任务类型至更通用的形状匹配或雕刻任务，以进一步测试策略的适应性和灵活性。
  3. 改进MPM算法的鲁棒性和分辨率，以支持对更复杂、真实场景的模拟，缩小仿真与现实的差距。

💡 一句话总结

本文提出TopoCut框架，旨在解决机器人对可变形物体进行多步切割时面临的拓扑变化复杂、状态感知困难、切割结果评估缺乏鲁棒性等核心挑战。其关键技术包括：1）基于粒子弹塑性求解器的高保真仿真环境，配备损伤驱动的拓扑发现机制；2）融合拓扑发现与拉普拉斯-贝尔特拉米特征分析的姿态不变光谱奖励模型；3）集成动态感知模块与粒子化分数熵离散扩散策略（PDDP）的策略学习流程。实验表明，该框架支持轨迹生成、可扩展学习与精确评估，并在多样物体几何、尺度、姿态与切割目标上表现出强泛化能力。
