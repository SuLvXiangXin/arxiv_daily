8-DoFs Cable Driven Parallel Robots for Bimanual Teleportation

arXiv: 2504.01554 作者: Cheng, Hung Hon, Hughes, Josie 日期: 2025/04/02

📝 详细解读

研究背景与动机

目前，直观机器人控制与模仿学习中的遥操作主要依赖几种主控制器：串联机器人（如运动学连杆）、并联平台和基于视觉的手势识别器。商业主控制器的工作空间从3自由度到最多7自由度不等。串联连杆设计存在连杆质量和自由度累积的问题，需要高性能执行器或轻质材料，且高扭矩执行器需要实时控制算法来补偿质量惯性。并联平台（包括缆驱并联机器人，CDPRs）虽然具有大工作空间、高有效载荷和刚性末端执行器等优点，但作为遥操作控制器的应用有限。现有CDPRs大多设计用于力反馈，或主要提供平移运动而姿态能力有限，其平移和姿态工作空间通常是耦合的，这增加了系统复杂性。此外，大多数商业遥操作控制器最多提供7个自由度（包括夹持器），而近期具有移动性的高自由度双手系统需要更多控制输入，但缺乏合适的直观主接口。

本文针对高自由度机器人遥操作中，现有主控制器自由度不足、工作空间受限以及平移与姿态控制耦合导致系统复杂的问题，提出了一种基于缆驱并联机器人（CDPRs）的新型低成本、高自由度主控制器。其核心思路是采用解耦的“3 + 3 + n”自由度结构：利用CDPR实现3自由度大范围平移，利用万向节机构实现3自由度姿态控制，并额外提供n个自由度用于夹持器和冗余关节控制，从而构建一个直观、可扩展的双手遥操作接口。

方法详解

系统整体为双手配置，每只手的主控制器由两个主要部分组成：一个由8个电机驱动的6自由度CDPR，以及一个作为其末端执行器安装的5自由度万向节机械手。在遥操作中，仅从CDPR获取平移运动，姿态完全由万向节处理，实现了平移与姿态控制的解耦。

图1：系统整体框架。两个8自由度缆驱并联机器人（CDPRs），每个配备一个基于万向节的控制器，用于为移动机械臂提供解耦的平移和旋转遥操作输入。

核心模块1：6自由度CDPR及其运动学模型
CDPR采用8缆驱动6自由度（3平移，3旋转）的构型。其运动学由缆线段 ABᵢ 构建，该线段从基座缆线附着点 Aᵢ⁰（相对于全局坐标系{0}）延伸到末端执行器缆线附着点 Bᵢ¹（相对于局部坐标系{1}）。具体表达式如论文公式(1)所示，其中 q_t = [x, y, z]ᵀ 表示平移，R₁⁰(q_o) 是基于欧拉角（XYZ序列）的旋转矩阵，表示姿态。在本文设计中，Aᵢ⁰ 被实现为固定点出口，简化了机械结构和运动学计算。

图2：CDPR运动学示意图。展示了全局坐标系{0}、末端执行器局部坐标系{1}、基座锚点 Aᵢ⁰、末端锚点 Bᵢ¹ 以及缆线段 ABᵢ。

核心模块2：万向节机械手
由于CDPR的姿态工作范围和力矩能力有限，在末端安装了一个万向节机械手来提供滚转、俯仰和偏航三轴旋转。其采用远程运动中心（RCM）设计，确保三个旋转轴相交于单个控制点，该点也与CDPR的平移位置 q_t 重合。此外，万向节集成了两个额外执行器以提供附加控制输入。其控制输出 q_m = [q₇, q₈, q₉, q₁₀, q₁₁]ᵀ ∈ ℝ⁵，其中 q₇, q₈, q₉ 分别代表滚转、俯仰、偏航角，q₁₀ 和 q₁₁ 代表触发器和一个旋转关节。这些附加关节的功能可由用户定义，例如分别用于夹持器角度控制和调整遥操作比例、控制冗余关节等。

图3：万向节机械手及其控制变量。展示了滚转（Roll）、俯仰（Pitch）、偏航（Yaw）三个旋转轴，以及触发器（Trigger）和旋转（Rotation）两个附加关节。

核心模块3：正向运动学与任务空间输出
由于万向节关节的低扭矩控制和反向驱动特性，末端执行器在大部分工作空间内保持零姿态。CDPR的正向运动学问题使用Levenberg-Marquardt（LM）优化方法求解，通过最小化实际缆线长度 l 与由姿态 q 计算出的长度 l_q(q) 之间的差异来估计 q_t。由于RCM设计，末端执行器的平移位置精度不受其姿态变化的影响，因此仅从正向运动学解算器中利用平移分量 q_t。

最终，主控制器的8自由度任务空间命令通过 X_m = [q_tᵀ - q_t,₀ᵀ, q_mᵀ]ᵀ ∈ ℝ⁸ 获得，其中 q_t,₀ 是用户启用任务空间命令时的参考位置（通常设在工作空间中心，姿态为0）。这种设计允许主控端与遥操作机器人的工作空间不同，平移命令 q_tᵀ 并非直接传递，而是结合遥操作机器人的初始位置 x_o，形成 X_s = [x_o + q_tᵀ - q_t,₀ᵀ, q_mᵀ]ᵀ，实现了平移控制的相对定位和姿态/关节控制的直接映射。

核心模块4：力反馈与虚拟 fixture
为了在末端姿态 q_o 可能显著增大的区域提醒操作者，系统实现了虚拟 fixture。将机器人的工作空间 𝒲 定义为末端姿态范数 ≤ 10° 的平移位置 q_t 集合。虚拟墙 𝒱 则定义为一个椭球体。当末端移动到虚拟墙外时，会产生一个指向工作空间中心的排斥力（力反馈），该力通过公式(9)和(10)映射为缆线张力 f（对应电机电流）。为了增强用户感知，还施加了高频振动。力反馈的目的是提示操作者重新定位参考点 q_t,₀，而非严格限制运动。

图4：遥操作系统的控制系统示意图。展示了从主端CDPR和万向节传感器，经过正向运动学（FK）解算和任务空间映射，到从端机器人控制器，以及从端传感器反馈至主端力反馈的完整闭环。

创新点总结
与现有方法相比，本文的创新点具体体现在：1) 解耦设计：通过CDPR专司平移、万向节专司姿态，解决了传统CDPRs平移与姿态工作空间耦合的问题，简化了控制。2) 可扩展的“6+n”自由度框架：在基础6自由度空间控制之上，通过万向节附加关节灵活支持夹持器、冗余关节或移动性控制，适应高自由度机器人需求。3) 低成本与轻量化实现：利用缆驱结构分担末端重量，实现低惯性和大工作空间，允许使用低成本执行器（如Dynamixel），降低了系统复杂性和成本。

实验与结果

本文通过一系列演示任务来验证系统的性能，未使用标准benchmark数据集。实验平台包括所开发的双臂CDPR主控制器，以及一个用于遥操作的8自由度机械臂（配备2自由度平移-倾斜云台相机）。对比的基线是传统的串联机器人臂作为主控制器的情况。

关键实验结果

  1. 工作空间扩展性：通过调整锚点，CDPR的工作空间可以轻松扩展。实验表明，仅通过调整上方锚点，工作空间体积可从约1.5 m³显著扩大。

  2. 平移与姿态精度：在虚拟墙定义的“近零姿态”工作空间（‖q_o‖ ≤ 10°）内，系统能实现精确的平移跟踪。与串联机器人主控制器相比，CDPR主控在相同平移位移下，引起的末端姿态偏差更小。

总结与启发

核心贡献：

  1. 提出并实现了一种新颖的、基于解耦设计的“3（CDPR平移）+ 3（万向节姿态）+ n（附加关节）”自由度缆驱并联机器人主控制器架构，有效解决了高自由度遥操作中控制输入不足和平移-姿态耦合的问题。
  2. 开发了首个低成本、轻量化的双臂CDPR遥操作系统原型，利用商业现货执行器和简单机械结构，展示了其在大工作空间、低惯性和高可扩展性方面的优势。
  3. 通过完整的运动学建模、任务空间映射、虚拟 fixture 力反馈设计以及真实的双手复杂任务演示，验证了该系统的可行性、精确性和实用性。

局限性：
论文自身提到的局限性包括：正向运动学求解的精度受缆线初始长度估计和锚点测量精度的影响；当前的力反馈依赖于执行器电流估计，未使用额外的张力传感器，精度有限；虚拟 fixture 的形状（椭球体）可能不是最优的，未来可以探索更复杂的工作空间边界定义。

对后续研究的启示：

  1. 设计范式：解耦的“平移+姿态+冗余”控制框架为设计高自由度、大工作空间的主控制器提供了新思路，可推广至其他并联或混合构型。
  2. 低成本实现：证明了利用CDPR的特性和低成本组件构建高性能遥操作接口的潜力，有助于降低先进机器人技术的应用门槛。
  3. 应用扩展：该系统特别适用于需要灵巧、冗余操作的野外或工业机器人遥操作，例如农业、搜救、太空作业等，为这些领域提供了直观的人机交互解决方案。未来的工作可以集中在提高力反馈精度、探索自适应虚拟 fixture 以及集成更智能的共享控制策略上。

💡 一句话总结

本文针对现有主控制器自由度有限、工作空间小的问题，提出一种基于缆绳驱动并联机器人的新型低成本高自由度主控制器。其核心技术采用“3+3+n”自由度结构：通过CDPR实现3自由度大范围平移，万向节机构提供3自由度朝向控制，外加n个自由度用于夹持器与冗余关节。该设计具有轻量化、大工作空间和低惯量特点。实验开发了首个双臂CDPR主控制器，成功遥操作8自由度机械臂完成拾放、打结等多种任务，实现了精确、通用的高自由度控制。
