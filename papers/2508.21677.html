Robust Convex Model Predictive Control with collision avoidance guarantees for robot manipulators

arXiv: 2508.21677 作者: Thomas B. Schön Team 日期: 2025-08-29

📝 详细解读

研究背景与动机

工业机械臂通常在杂乱环境中运行，安全运动规划至关重要。当前主流方法采用解耦的流水线：首先使用基于采样的规划器寻找无碰撞路径，然后通过时间缩放生成动力学可行的轨迹，最后使用基于精确模型的反馈线性化进行轨迹跟踪。该方法虽高效，但存在关键局限性：1）严重依赖精确的建模，而模型参数不确定性普遍存在；2）由于路径规划仅在认证的路径上保证无碰撞，因此控制器必须严格跟踪该路径，缺乏灵活性；3）为确保在模型失配下的安全，实践中通常限制运行速度，牺牲了效率。本文针对在模型不确定下实现既快速又安全的运动规划这一痛点，提出了一种结合鲁棒控制与走廊式规划的新视角。其核心思路是：利用反馈线性化获得线性预测模型并推导状态/输入依赖的模型误差上界，设计鲁棒管模型预测控制（MPC）来优化管的大小以减少保守性；同时，利用学习得到的符号化构型距离函数（SCDF）生成构型空间中的无碰撞“走廊”，从而将鲁棒管约束在无碰撞区域内，最终形成一个可快速求解的凸MPC问题。

方法详解

整体框架是一个在线的、带鲁棒性保证的凸模型预测控制器。输入为当前机械臂状态（关节位置与速度）、目标状态以及环境障碍物信息，输出为经过反馈线性化后的关节加速度指令。该框架的核心在于同时优化一条名义轨迹和一个围绕该轨迹的椭球“管”（Tube），确保在模型不确定下真实系统状态始终位于管内，并通过将整个管约束在由SCDF定义的无碰撞“走廊”内来保证避障。

图1：方法整体框架示意图。机械臂（红色网格）需要从起点（白色网格）运动到终点（绿色网格）并避开障碍物（灰色球体）。红色半透明轨迹表示闭环运动轨迹，其周围的管状区域由鲁棒MPC保证，确保了在模型误差下的安全。

核心模块一：反馈线性化与模型误差上界。对具有参数不确定性的机械臂动力学模型（公式1）应用基于名义模型的反馈线性化（公式2），得到等效的双积分器系统加一个模型误差项 Δ_θ (公式3,4)。关键创新在于推导了一个状态和输入依赖的误差范数上界 β(x,a) = a||a|| + b||q̇|| + c (公式6)，其中常数a,b,c通过对不确定参数集Θ和状态空间采样得到（公式7-9）。这与现有方法中使用恒定（最坏情况）误差界相比，显著减少了保守性。

核心模块二：辅助控制器与管动力学。设计一个线性辅助控制器 a = ā + K(x - x̄) (公式10)，其中K和Lyapunov矩阵P离线计算，满足收缩条件（公式11）。定义了一个由P和缩放因子δ控制的椭球集 ℰ(δ)（公式12）。核心命题2给出了管尺度的递推关系：δ_+ = (ρ + dL_β)δ + dβ(x̄, ā) (公式13)。这确保了若当前时刻真实状态与名义状态的偏差在δ内，则下一时刻的偏差可由δ_+界定。通过在线优化δ，可以动态调整管的“粗细”。

核心模块三：凸加速度约束。反馈线性化将原始扭矩约束u∈𝒰映射为非凸的加速度约束。本文提出一种采样方法：从一个加速度盒子约束𝒜开始，通过在状态空间𝒳中均匀采样，并检查每个采样状态和𝒜顶点是否违反扭矩约束（公式17）；若违反，则均匀缩小𝒜，迭代直至满足所有采样点，从而得到一个保证扭矩约束的凸加速度约束集𝒜。

核心模块四：鲁棒凸MPC问题构建。最终的优化问题（公式18）决策变量为名义状态轨迹X̄、名义加速度Ā和管尺度序列δ。目标函数（18a）驱动名义轨迹趋向一个作为人工参考的终端稳态，并最小化控制量和管尺度。约束包括：初始管约束（18b）、名义线性动力学（18c）、终端零速度与稳态管约束（18d）、根据命题2递推的管尺度动力学（18e），以及通过Pontryagin差集紧缩处理的状态和输入约束（18f, 18g）。该问题是一个凸二阶锥规划，可高效求解。定理1保证了闭环系统的递归可行性、约束满足和收敛性。

实验与结果

实验在仿真环境中进行，使用一个6自由度工业机械臂模型。基准方法包括：1）标准的“规划-跟踪”流水线（Path+Track）；2）使用恒定（最坏情况）误差界的鲁棒MPC（Worst-Case Tube MPC）；3）不考虑鲁棒性的凸MPC（Nominal MPC）。关键评估指标为任务成功率和运动时间。

图3：在不同不确定性水平下，成功完成任务的平均运动时间。本文方法（蓝色）在保证高成功率的同时，运动时间最短，体现了快速性优势。

图4：消融实验。对比了完整方法、不使用状态/输入依赖误差界（用恒定界）、以及不使用走廊规划（即严格路径跟踪）的性能。结果表明，两个核心组件（自适应误差界和走廊规划）对在高不确定性下维持高性能都至关重要。

总结与启发

本文的核心贡献在于：1）首次为6自由度非线性不确定机械臂动力学构建了一个保证避障的凸MPC公式，实现了快速求解；2）推导了状态和输入依赖的模型误差上界，并集成到管MPC中，相比恒定误差界减少了保守性；3）提出了结合学习SCDF的走廊规划方法，将鲁棒管约束在无碰撞区域内，替代了严格的路径跟踪，提供了更大的灵活性。

论文提到的局限性包括：学习SCDF需要针对特定机械臂和障碍物类别的数据；稳态管大小δ_f（公式16）可能限制目标点的可达区域；当前方法假设状态完全可测。

这项工作对后续研究的启示：将学习（SCDF）与基于模型的鲁棒控制（管MPC）相结合是一条有前景的路径；所提出的状态/输入依赖误差界方法可推广到其他具有结构化不确定性的系统；凸优化框架为在机器人上实现实时、安全的复杂行为控制提供了新的工具。

💡 一句话总结

本文针对工业机器人在模型不确定的杂乱环境中，如何实现快速且安全的运动规划这一核心问题，提出了一种鲁棒凸模型预测控制（MPC）方法。关键技术包括：1）基于反馈线性化和模型误差上界的鲁棒管MPC，以优化控制保守性；2）利用有符号配置距离函数（SCDF）生成构型空间中的无碰撞区域，从而构建凸的避障约束。在6自由度工业机器人的仿真实验中，该方法在更高模型不确定性水平下优于基准方法，并实现了更快的运动速度。
