Emergent World Representations in OpenVLA

arXiv: 2509.24559 作者: Molinari, Marco, Nevali, Leonardo, Navani, Saharsha, Younis, Omar G. 日期: 2025/09/29

📝 详细解读

研究背景与动机

在机器人领域，强化学习（RL）方法主要分为两类：基于模型的方法显式学习环境动力学以辅助决策，而无模型方法（如基于策略的RL）则直接从经验中学习策略，不显式建模环境。视觉-语言-动作模型（VLAs）是使用无模型RL训练的Transformer模型，已在多种现实应用中展现出潜力。然而，一个核心问题悬而未决：这些仅通过策略学习复杂行为的VLAs，是否在内部隐含地学习了世界模型（即环境动态知识）？这是基于模型RL的标志性特征。本文针对这一具体痛点，旨在探究当前最先进的VLA模型——OpenVLA，是否在其内部激活中编码了潜在的世界模型。本文的核心思路是：利用嵌入算术定义状态转移向量，并通过训练线性/非线性探针（Probe）从模型内部激活中预测该向量，以此作为存在隐含世界模型的证据。

方法详解

本文的整体方法论是探测分析。其核心是验证一个函数 ( f: \mathbf{a}t \mapsto \Delta\mathbf{e}{t\rightarrow t+K} ) 的存在性，该函数能从模型在时间 ( t ) 的内部激活 ( \mathbf{a}t ) 中，预测出执行动作后 ( K ) 步的环境嵌入变化量 ( \Delta\mathbf{e}{t\rightarrow t+K} = \mathbf{e}_{t+K} - \mathbf{e}_t )。这等价于在嵌入空间中学习一个状态转移向量。

图1：利用OpenVLA涌现世界模型的可解释性规划管道。管道截取模型策略的内部激活 ( a_t )，通过线性探针预测状态转移向量 ( \Delta\hat{e}_{t\to t+K} )，随后使用稀疏自编码器（SAE）将其分解为稀疏的、有意义的特征向量，从而实现人工验证。

理论基础：研究首先在Koopman算子理论框架下进行论证。对于一个由最优策略诱导的闭环动力系统 ( e_{t+1} = F(e_t) )，论文证明在一定的假设下（如遍历性、有界性等），通过使用模型激活作为观测函数，其对应的有限维Koopman算子估计量 ( \widehat{\mathcal{K}}_{N,M}^K ) 可以强收敛到真实的Koopman算子 ( \mathcal{K}^K )。这意味着，从理论上讲，通过探测一个最优智能体的激活，可以恢复其未来的状态。这为后续的实证探测方法提供了理论动机。

核心模块与技术细节：

  1. 数据与特征提取：研究基于OpenVLA模型，使用LIBERO数据集的四个子集（goal, spatial, long, object）。从模型残差流（residual stream）的第7、15、22、30层提取激活 ( \mathbf{a}t )。目标标签 ( \Delta\mathbf{e}{t\rightarrow t+K} ) 则通过计算连续时间步的嵌入（对图像块token进行平均池化后得到）差值获得。时间跨度 ( K ) 取值为1、3、10、30。
  2. 探针训练：
    • 线性探针：使用带L1正则化的Lasso回归进行训练，损失函数为 ( \mathcal{L} = \left|f(\mathbf{a}t) - \Delta\mathbf{e}{t\rightarrow t+K}\right|^{2} + \lambda\left|\boldsymbol{\beta}\right|_{1} )。通过网格搜索优化超参数。
    • MLP探针：为了检验线性表示假说（LHR）在本场景下的适用性，同时训练了多层感知机探针。
  3. 基线设置：为规避探针分析可能存在的相关性而非因果性问题，论文设置了关键基线：在原始嵌入 ( \mathbf{e}_t ) 上训练相同的探针。如果基于内部激活 ( \mathbf{a}_t ) 的探针性能显著优于基于原始嵌入 ( \mathbf{e}_t ) 的基线，则更有力地表明预测能力源自模型内部的计算（即世界模型），而非探针自身从简单输入中学习到了转移规律。

创新点：与现有工作（如在文本游戏或棋盘游戏中探测世界模型）相比，本文的创新在于将世界模型的探测对象扩展到了大规模、多模态的机器人VLA模型；并提出了一个结合探针与稀疏自编码器（SAE）的可解释性规划管道雏形（图1），为将隐含的世界模型转化为可理解、可验证的预期提供了思路。

实验与结果

实验设置：

  • Benchmark/数据集：LIBERO数据集（包含goal, spatial, long, object四个子集），共计400个episodes，66931个时间步。
  • 模型：OpenVLA（7B参数），及其早期检查点（v01）和在LIBERO上微调后的版本。
  • Baseline方法：基于原始嵌入 ( \mathbf{e}_t ) 训练的探针（线性/MLP）。
  • 评估指标：回归任务的 ( R^2 ) 分数，并使用置换检验（Permutation Test）计算p值评估统计显著性。

关键实验结果：

  1. 激活探针显著优于嵌入基线：如图2所示，在所有时间跨度 ( K ) 和所有数据集上，基于模型内部激活 ( \mathbf{a}_t ) 的最佳探针的 ( R^2 ) 分数均一致且显著地高于基于原始嵌入 ( \mathbf{e}_t ) 的基线探针。绝大多数置信区间没有重叠，表明统计显著性。

图2：在OpenVLA上，基于嵌入的探针与基于激活的最佳探针的 ( R^2 ) 对比（含置信区间）。结论：基于激活的探针性能全面超越基于嵌入的探针，且大多数置信区间不重叠，凸显了统计显著性。

  1. 统计显著性验证：对123个在OpenVLA上 ( R^2 > 0 ) 的探针进行的置换检验全部成功（p < 0.01），综合p值 < 0.0001（表1），强有力地拒绝了“探针没有预测能力”的原假设。
  2. 世界模型的涌现与训练规模相关：通过比较不同模型检查点，发现世界模型的预测能力随着训练计算量（规模）的增加而涌现。在早期检查点v01中，几乎没有发现世界模型的证据（图4）；而完整的OpenVLA则表现出强大的预测能力。直接在LIBERO上微调OpenVLA反而削弱了这种证据（图5），突显了大规模预训练对泛化能力及世界模型涌现的重要性。

图4：OpenVLA早期检查点（v01）的探针性能对比。可见其激活探针性能与嵌入基线差异不大，表明世界模型尚未形成。

图5：在LIBERO上微调后的OpenVLA探针性能对比。性能相比原始OpenVLA有所下降，表明狭义的任务微调可能不利于世界模型的保持或泛化。

  1. 世界模型在模型中层最易探测：如图3所示，探测状态转移知识的最佳位置是模型的中间层（如第15、22层）。更深层（如第30层）的效果较差，这与先前研究认为深层更关注与输出（logits）相关信息相符。

图3：OpenVLA在不同层和不同K值上的测试 ( R^2 ) 分数（含插值）。结论：世界模型的知识主要集中在OpenVLA的中间层。

  1. 线性表示假说的证据：比较线性探针与MLP探针的性能（表2），发现MLP探针从未显著优于线性探针（在48次比较中，0次获胜），而线性探针在多数情况下显著更优。这为线性表示假说（LHR）在本场景下提供了证据，也支持了后续使用线性SAE进行特征分解的可行性。

总结与启发

核心贡献：

  1. 实证发现：首次通过严谨的探测方法证明，使用基于策略的RL训练的大规模VLA模型（OpenVLA）内部编码了关于环境状态转移的潜在知识，即“涌现的世界模型”。
  2. 特性分析：揭示了该世界模型的涌现与预训练计算规模正相关，且其知识主要分布在模型的中间层，为模型的可解释性分析提供了重要线索。
  3. 应用蓝图：勾勒了一个结合线性探针与稀疏自编码器（SAE）的“可解释性规划”管道，为未来构建可验证、可信赖的机器人决策系统提供了思路。

局限性：

  1. 平均池化限制：由于对嵌入和激活进行了平均池化（得到全局向量），无法将预测的状态转移向量解耦到图像块（patch）级别，从而限制了进行块级干预或精细可解释性分析的能力。
  2. 计算资源限制：实验仅在总计400个episode的数据上进行，更全面的分析（如逐块探针训练）需要更大的计算资源。

对后续研究的启示：

  1. 机理探索：此工作为理解VLAs如何以及为何能有效工作打开了新视角，后续可深入探究世界模型的形成机制、其与策略性能的关联等。
  2. 可解释性与安全：提出的可解释性管道蓝图指向了一个重要方向：通过提取和解释模型内部的“预期”，实现对人机协作中关键决策的事前验证或否决，对高风险应用（如医疗辅助）尤为重要。
  3. 方法扩展：未来的工作可以致力于开发非平均池化的探测方法，实现更细粒度的状态转移分析和干预，从而更深入地理解和利用VLA中涌现的世界模型。

💡 一句话总结

本文研究当前先进的视觉语言动作模型OpenVLA在基于策略的强化学习训练中，是否隐式地学习了环境动态的世界模型。通过嵌入算术和线性/非线性探针技术，分析模型内部激活以探测状态转换向量。核心实验发现，探针在模型中间层激活上对状态转移的预测能力显著超过基线（仅使用嵌入），表明OpenVLA确实编码了内部世界模型，且该模型在训练过程中涌现。
