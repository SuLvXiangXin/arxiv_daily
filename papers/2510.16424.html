Learning to Optimize Edge Robotics: A Fast Integrated Perception-Motion-Communication Approach

arXiv: 2510.16424 作者: Chengzhong Xu Team 日期: 2025-10-18

📝 详细解读

研究背景与动机

边缘机器人系统使资源受限的移动机器人能够将计算密集型任务卸载到边缘服务器。其核心挑战在于如何在有限的机器人功率下传输海量多模态数据流。现有主流方法主要关注通信吞吐量最大化或用户公平性，但这些指标并非为机器人任务量身定制，无法考虑不同模态和帧间的数据多样性。近期提出的多模态通信方法通过传输压缩的传感器数据并动态调整压缩比来应对，但它们普遍忽略了机器人功能与通信条件之间的相互依赖性。例如，如果机器人没有运动，则完全不需要通信。此外，这些方法往往忽视跨模态的资源分配，可能导致传感器数据失真或过度的通信开销。

本文针对上述痛点，提出了集成感知、运动与通信的新视角。核心思路是：通过联合优化机器人的压缩比、下采样率和发射功率，使其能够根据感知场景变化率和机器人运动状态动态调整通信策略，将资源优先分配给变化更剧烈的场景，从而在接收端促进数据恢复并提升保真度。

方法详解

本文提出的方法包含两个核心阶段：首先将非凸的集成优化问题转化为可精确求解的几何规划形式；其次，利用学习优化范式，训练一个深度神经网络来模仿几何规划求解器的行为，以实现实时、低复杂度的决策。

图2：LTO-IPMC算法流程。包含三个阶段：1）离线生成训练样本：使用GP算法求解不同输入下的最优策略，生成数据集；2）离线训练DNN模型：使用生成的数据集训练一个全连接网络，学习从系统状态到最优策略的映射；3）在线决策：部署训练好的DNN，机器人实时估计系统状态并输入DNN，即时获得通信策略。

整体问题建模：系统模型如图1a所示，包含一个移动机器人和一个边缘服务器。任务时长被划分为H个子时段，每个子时段包含L个数据帧。机器人在第h个子时段、第l个时隙的状态为 ( \mathbf{s}{h,l} )，包含位置和朝向。机器人的动作（线速度和转向角）通过阿克曼运动学模型驱动状态演化。机器人的测量数据包含M种模态（如相机、激光雷达），每种模态的数据可进行下采样（比率 ( d{m,h} )）和压缩（比率 ( c_{m,h} )）后，以功率 ( p_{m,h} ) 通过无线信道传输至服务器，需满足数据率约束和总平均功率预算约束。

核心模块一：感知感知的目标函数。目标是最小化所有子时段和模态中最差的数据失真。由于失真函数依赖于服务器端的恢复数据（未知），本文提出了场景变化指标作为代理。SCI ( Q_{m,h} ) 通过计算连续帧间数据的均方误差来衡量场景变化率。将其归一化得到 ( \widetilde{Q}{m,h} )。基于此，构建了感知感知的代理目标函数 ( C_0(\mathbf{D},\mathbf{C}) := \max{m,h} \tilde{Q}{m,h}^{\alpha}{d{m,h}^{-1}c_{m,h}^{-1}} )，其中 ( \alpha ) 为调优参数。该模型与真实数据匹配良好（图1b），例如在机器人转向时 ( \widetilde{Q} ) 值高，在静止时 ( \widetilde{Q} ) 值低。

核心模块二：运动感知的通信约束。为确保在机器人转向等高不确定性运动时的数据保真度，需要避免在转向点进行下采样和压缩。通过模型预测控制预测未来机器人状态轨迹，并计算轨迹曲率 ( \theta_h )。设定曲率阈值 ( \theta_{\mathrm{th}} )，当 ( \theta_h \geq \theta_{\mathrm{th}} ) 时，认为该子时段为转向路径，并施加更严格的下采样和压缩比率下限约束（公式10a, 10b）。该约束能有效检测出转向路径（图1c）。

问题形式化与转化：联合优化下采样率、压缩率和发射功率的IPMC问题 ( \mathsf{P} ) 是非凸的。通过变量替换 ( \widehat{d}{m,h}=\ln d{m,h} ) 和 ( \widehat{c}{m,h}=\ln c{m,h} )，将目标函数和约束进行对数变换，最终将原问题等价转化为一个凸的几何规划问题 ( \mathsf{P}_1 )，可使用内点法最优求解，但其计算复杂度为 ( \mathcal{O}((3MH)^{3.5}) )，对于大规模问题实时性差。

核心模块三：学习优化加速。为解决GP求解速度慢的问题，提出了LTO-IPMC。其核心是训练一个DNN来模仿GP求解器的输入-输出映射。如图2所示：1）离线样本生成：使用GP算法求解大量随机生成的系统状态 ( {\mathbf{Z}, \widetilde{\mathbf{Q}}, \bm{\psi}} ) 对应的最优策略 ( {\mathbf{D}^{}, \mathbf{C}^{}, \mathbf{P}^{*}} )，构成数据集。2）离线DNN训练：采用一个五层全连接网络（隐藏层宽度依次为64, 128, 256, 512，使用ReLU激活函数），以均方误差为损失函数，Adam为优化器，学习上述映射关系。3）在线决策：将训练好的轻量级DNN部署于机器人，实时估计系统状态并输入网络，前向传播即可瞬间获得近似最优的通信策略，实现实时决策。

创新点：1) 框架创新：首次明确将感知（场景变化率）、运动（轨迹曲率）与通信（采样、压缩、功率）进行联合建模与优化。2) 方法创新：提出了基于SCI的感知代理目标和基于MPC预测的运动感知约束。3) 效率创新：采用LTO范式，将耗时的迭代优化转化为一次前向传播，实现了超过10倍的计算加速。

实验与结果

实验设置：实验在Ubuntu工作站上进行。使用LIMO机器人采集了总计10137秒的数据集，包含相机（RGB图像）和激光雷达（点云）两种模态。整个任务被划分为H=10137个1秒长的子时段，每个子时段L=7帧。通信带宽为1 MHz，噪声功率为-80 dBm，信道增益服从均匀分布。对比的基线方法包括：MaxRate（最大化总速率）、Fairness（最大化最小时隙帧率）和STS（一种自适应的视频传输方案）。

关键定量结果：

  1. 参数敏感性分析：如图3a-b所示，调整感知目标中的参数 ( \alpha ) 发现，( \alpha =1 ) 时能取得最佳的PSNR和最差的MSE性能，表明需要权衡感知指标的权重。

    图3a-b：展示了感知目标函数中调优参数α对图像PSNR和激光雷达MSE的影响。存在一个最优的α值（=1），使性能最佳。

  2. 运动约束阈值分析：如图3c-d所示，运动感知约束中的曲率阈值 ( \theta_{\mathrm{th}} ) 显著影响系统性能，( \theta_{\mathrm{th}} =0.07 ) 能取得最佳平衡，验证了运动约束的必要性和阈值选择的重要性。

    图3c-d：展示了运动感知约束中曲率阈值θ_th对性能的影响。θ_th=0.07时性能最优，与图1c中正确检测转弯路径的阈值一致。

  3. 与基线方法对比：如图3e-f所示，提出的GP-IPMC方法在所有方案中取得了最高的PSNR值和最低的MSE值。具体而言，在图像PSNR上，GP-IPMC相比次优方法至少提升了4.03%；在激光雷达MSE上，相比次优方法提升了8.69%。MaxRate和Fairness方法由于忽略多模态平衡，性能较差。

    图3e-f：GP-IPMC与三种基线方法在图像PSNR和激光雷达MSE上的对比。GP-IPMC全面优于所有基线。

定性结果：图4可视化了不同方法重建的相机和激光雷达数据。GP-IPMC重建的图像和点云与原始数据最为接近。MaxRate和Fairness方案导致图像最模糊。STS方案改善了图像质量，但严重劣化了激光雷达数据（红色框标记处有点缺失），且图像文本存在明显差异。

图4：不同方法重建数据的可视化对比。GP-IPMC在两种模态上都保持了最高的保真度。

LTO-IPMC性能验证：

  1. 学习效果：如图5a-b所示，训练的DNN在测试集上达到了0.98的R-squared精度和0.003的损失，表明其成功学会了模仿GP求解器。

    图5a-b：LTO-IPMC中DNN的训练与测试精度、损失曲线，表明模型得到了有效训练且泛化良好。

  2. 性能接近性：如图5c-d所示，LTO-IPMC的MSE和PSNR性能略差于GP-IPMC，但差距很小，在可接受范围内。

    图5c-d：LTO-IPMC与GP-IPMC在MSE和PSNR性能上的对比，LTO-IPMC取得了接近最优的性能。

  3. 计算效率：如图5e所示，LTO-IPMC的计算时间不到GP-IPMC的10%，且随着问题规模增大，加速效果更显著。LTO-IPMC的单次推理时间仅需4毫秒，具备实时执行能力。

    图5e：GP-IPMC与LTO-IPMC的计算时间对比。LTO-IPMC实现了超过10倍的加速，且耗时几乎不随样本数增加。

消融实验总结：实验通过对参数 ( \alpha ) 和 ( \theta_{\mathrm{th}} ) 的调整，分别验证了感知感知目标函数和运动感知约束组件对最终性能的关键影响，二者均为IPMC框架的有效组成部分。

总结与启发

核心贡献：

  1. 提出了集成感知、运动与通信的IPMC框架，首次将机器人本体的感知场景动态和运动状态显式地纳入通信资源优化中，实现了机器人任务导向的自适应传输。
  2. 设计了基于学习优化的高效求解方法LTO-IPMC，通过深度神经网络模仿复杂优化求解器，在保持接近最优性能的同时，将计算复杂度降低了超过一个数量级，满足了边缘机器人的实时性要求。

局限性：论文自身提到，精确的GP求解方法在变量维度高时（( \mathcal{O}((3MH)^{3.5}) )）计算耗时，不适合直接在线使用。而LTO方法需要离线生成大量的训练数据，且其性能依赖于训练数据的覆盖范围和DNN的泛化能力。

对后续研究的启示：

  1. 联合优化范式：证明了将高层机器人任务语义（感知、运动）与底层通信参数进行联合优化的可行性与优越性，这一范式可扩展至更复杂的机器人任务（如抓取、导航）与网络资源调度中。
  2. 学习优化应用：展示了LTO在解决机器人系统中复杂、实时优化问题上的巨大潜力，可应用于其他具有类似复杂约束和实时性要求的控制与决策问题。
  3. 模态平衡：研究揭示了在多模态机器人系统中进行跨模态资源分配的重要性，为未来异构机器人传感器网络的协同传输提供了思路。

💡 一句话总结

本文针对边缘机器人系统中机器人功能与通信条件相互依赖、通信开销过大的核心问题，提出了一种快速集成感知-运动-通信（IPMC）方法。该方法通过联合优化压缩比、传输频率和发射功率等通信策略，动态适配机器人的感知与运动状态，从而减少冗余数据传输。关键技术是采用了学习优化（LTO）范式，设计了一个模仿学习神经网络来近似求解优化问题。实验表明，该神经网络的计算复杂度比现有最优求解器降低了10倍以上，并证明了IPMC的优越性与LTO的实时执行能力。
