Collision-Free Trajectory Planning and control of Robotic Manipulator using Energy-Based Artificial Potential Field (E-APF)

arXiv: 2508.07323 作者: Manoranjan Sinha Team 日期: 2025-08-10

📝 详细解读

研究背景与动机

机器人轨迹规划在动态、杂乱环境中是一项关键挑战，需要同时满足时间效率、运动平滑性和驱动约束。传统路径规划方法，如人工势场法（APF），因其计算效率和实时调整能力而被广泛应用。然而，传统APF依赖基于位置的势函数，存在局部最小值问题，并且在障碍物附近由于牛顿力学原理会产生振荡运动。针对这些局限性，本文提出了一种新的能量基人工势场（E-APF）框架。该框架的核心思路是摒弃传统的牛顿力学视角，转而采用拉格朗日力学，将位置和速度依赖的势函数相结合，从而动态适应环境、规避局部最小值，并结合轨迹优化确保生成平滑、时间最优的轨迹。

方法详解

提出的整体框架是一个模块化结构，集成了路径规划、轨迹优化和运动控制。

图2：所提方法的整体架构。首先，人工势场（APF）路径规划算法考虑目标吸引和障碍物排斥，生成无碰撞的参考路径。随后，时间最优和最小急动度轨迹优化模块对参考路径进行精细化处理，确保运动平滑且动态可行。优化后的轨迹作为输入传递给计算力矩控制器（CTC），该控制器补偿Kinova Gen3机械臂的非线性动力学并生成关节扭矩。最后，被控对象模型执行运动并提供闭环控制所需的反馈。

核心模块一：能量基人工势场（E-APF）
E-APF是本文的核心创新点，旨在解决传统APF的局部最小值和振荡问题。它通过拉格朗日力学进行建模，定义了包含动能和势能的拉格朗日量。

  1. 吸引势：定义了静态吸引势 (U_a = \frac{1}{2}k_a r_e^2) 和动能吸引势 (K_a = \frac{1}{2}\gamma k_a v_e^2)，其中 (r_e) 和 (v_e) 分别是机器人相对于目标的位置和速度，(\gamma \in (0,1)) 是速度缩放因子。
  2. 排斥势：定义了静态排斥势 (U_r = \frac{1}{2}k_r (\frac{1}{r_o} - \frac{1}{\rho_o})^2) 和动能排斥势 (K_r = \frac{1}{2}\gamma k_r (\frac{1}{v_o} - \frac{1}{\mu_o})^2)，其中 (r_o) 和 (v_o) 是相对于障碍物的位置和速度，(\rho_o) 和 (\mu_o) 分别是空间和速度影响范围。
  3. 力计算：分别构建吸引和排斥的拉格朗日算子 (L_a = K_a - U_a) 和 (L_r = K_r - U_r)，然后通过求解欧拉-拉格朗日方程 (\frac{d}{dt}(\frac{\partial L}{\partial v}) - \frac{\partial L}{\partial r} = F) 来推导出力，而不是传统APF中直接对势能求梯度。最终得到的总E-APF力为：
    [
    F_{e-apf} = \gamma k_a \dot{v}_e - k_a r_e + \gamma k_r \left(\frac{3}{v_o} - \frac{2}{\mu_o}\right)\frac{\dot{v}_o}{v_o^3} + k_r \left(\frac{1}{r_0} - \frac{1}{\rho_0}\right)\left(\frac{1}{r_0}\right)^2
    ]
    该力公式同时包含了位置和速度（及其导数）项。即使位置相关的吸引力和排斥力达到平衡（传统APF的局部最小值情况），速度相关项也能确保机器人继续向目标运动，从而有效避免了局部最小值陷阱和振荡行为。

核心模块二：混合轨迹优化
E-APF生成的路径经过一个轨迹优化模块，旨在计算动态可行、平滑且时间高效的关节空间轨迹。优化问题定义如下：

  • 目标函数：(\min_{\mathbf{q}(t),T_f} \int_{0}^{T_f} |\dddot{\mathbf{q}}(t)|^2 dt + \lambda T_f)。第一项最小化积分平方急动度以确保运动平滑性，第二项是对总执行时间 (T_f) 的惩罚，(\lambda > 0) 用于权衡平滑性与速度。
  • 约束条件：包括关节速度约束（-10 ≤ (\dot{\mathbf{q}}(t)) ≤ 10 rad/s）和关节加速度约束（-50 ≤ (\ddot{\mathbf{q}}(t)) ≤ 50 rad/s²），以保证动态可行性并保护硬件。

核心模块三：计算力矩控制（CTC）
为了精确跟踪优化后的期望轨迹 ((q_d, \dot{q}_d, \ddot{q}_d))，采用计算力矩控制器来补偿机器人的非线性动力学。控制律为：
[
\tau = M(q)[\ddot{q}_d + k_p q_e + k_d \dot{q}_e] + C(q, \dot{q})\dot{q} + G(q)
]
其中 (q_e = q_d - q)， (\dot{q}_e = \dot{q}_d - \dot{q}) 是跟踪误差，(k_p, k_d > 0) 为PD增益。该控制律通过反馈线性化，将系统误差动态简化为线性二阶系统，确保指数收敛。

实验与结果

实验设置：在仿真中使用7自由度Kinova Gen3机械臂进行验证。起始构型为 ([0,0,0,0,0,0,0])，目标构型为 ([1.5,-0.8,0.5,-0.9,0.6,0.3,-0.4])。环境中设置了三个障碍物：一个圆柱体（高0.15m，半径0.075m）和两个球体（半径均为0.1m）。
对比方法：将提出的E-APF方法与传统APF方法进行对比。
关键参数：如下表所示。

参数 APF值 E-APF值
吸引增益 (k_a) 5.0 5.0
排斥增益 (k_r) 10.0 10.0
障碍物影响距离 (\rho_0) 0.4 m 0.4 m
速度缩放因子 (\gamma) - 0.8

实验结果：

图3：APF（蓝色）与E-APF（红色）在三维空间中的轨迹对比。传统APF轨迹在障碍物附近表现出多次急转弯和振荡行为，而E-APF轨迹明显更平滑，方向流一致，且与障碍物保持了更安全的距离。

  • 轨迹平滑性与时间：E-APF产生了更平滑、无振荡的轨迹。传统APF到达目标用时2.0426秒，E-APF用时2.1216秒。E-APF用时稍长，论文指出这是为获得改进的轨迹平滑性所付出的合理代价。

图4：E-APF方法下七个关节的角度随时间演化图。所有关节角度变化平滑，无突变，这是混合轨迹优化框架最小化急动度的直接结果。

  • 相对于目标的位置与速度：

图5：末端执行器相对于目标的位置范数。E-APF（红色）的曲线下降更迅速，表明其收敛到目标的速度比传统APF（蓝色）更快。

图6：末端执行器相对于目标的速度范数。传统APF呈现多个尖峰，表明存在剧烈的加减速（急动运动）。E-APF的速度曲线峰值更高但更平缓，表明其动态避障策略更为主动和流畅。

  • 相对于障碍物的位置与速度：

图7：末端执行器相对于最近障碍物的位置范数。在整个轨迹中，E-APF（红色）比传统APF（蓝色）与障碍物保持了更大的距离，尤其是在0.75-1.25秒和1.75秒附近，安全性更高。

图8：末端执行器相对于最近障碍物的速度范数。传统APF在约1.25秒处出现极高的尖峰（>11 m/s），可能导致不稳定或急动运动。E-APF的峰值出现在不同的时间点（如1.5秒后），且在靠近障碍物时（约1.25秒）相对速度较低，表现出更优的预测性避障行为。

总结与启发

核心贡献：

  1. 提出E-APF框架：通过引入基于拉格朗日力学的能量视角，将位置和速度同时纳入势函数，从根本上解决了传统APF的局部最小值和障碍物附近振荡问题。
  2. 集成混合轨迹优化：将E-APF与一个联合最小化急动度和执行时间的优化器相结合，在满足关节速度/加速度约束的前提下，生成了既平滑又时间高效的动态可行轨迹。

局限性：论文自身提到，当前工作是在仿真环境中验证的。未来的工作需要将该框架扩展到多机器人协调以及包含动态移动障碍物的实时适应环境中，并最终部署到物理硬件上。

后续研究启示：E-APF所采用的能量（拉格朗日）框架为机器人路径规划提供了一个新的、更具物理一致性的建模思路。这种将动力学特性（速度）直接嵌入势场的方法，对于需要在动态环境中进行实时、平滑、安全导航的复杂机器人系统（如高自由度机械臂、移动机器人集群）具有重要的借鉴意义。将此类方法与模型预测控制（MPC）或学习-based方法结合，可能是进一步提升其在高度不确定环境中性能的有前途的方向。

💡 一句话总结

本文针对动态杂乱环境中机器人轨迹规划存在局部最小值和运动振荡的问题，提出一种基于能量的人工势场方法。该方法整合位置与速度依赖的势函数，并与混合轨迹优化器结合，在速度与加速度约束下共同最小化加加速度和执行时间。在7自由度Kinova Gen3机械臂上的仿真验证表明，该方法能生成无碰撞、平滑、时间高效且无振荡的轨迹。
