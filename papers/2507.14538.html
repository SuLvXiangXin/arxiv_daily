A 21-DOF Humanoid Dexterous Hand with Hybrid SMA-Motor Actuation: CYJ Hand-0

arXiv: 2507.14538 作者: Chai, Jin, Yao, Xiang, Hou, Mengfan, Li, Yanghong, Dong, Erbao 日期: 2025/07/19

📝 详细解读

研究背景与动机

当前，机器人灵巧手是实现复杂、拟人化操作任务的关键部件。传统灵巧手主要依赖电机驱动，虽能提供精确控制，但往往导致系统笨重、体积庞大，且缺乏柔顺性，难以进行安全的物理交互。其他驱动方式如气动或液压虽能提供良好的柔顺性与功率密度，但需要复杂的外部泵和管路，限制了移动性。形状记忆合金（SMA）作为一种新型智能材料驱动器，具有高功率密度、柔顺性和静音特性，但其响应速度慢、热管理复杂，单独构建完整灵巧手面临挑战。本文针对构建紧凑、强健且具备良好交互柔顺性的仿人灵巧手这一痛点，提出了一种混合SMA与电机驱动的新视角。其核心思路是：利用SMA的柔顺性和高功率密度驱动手指关节，实现轻量化与被动柔顺；同时利用传统电机驱动拇指基座关节，弥补SMA响应慢的不足，共同构建一个21自由度的仿人灵巧手（CYJ Hand-0）。

方法详解

本文提出的CYJ Hand-0是一个具有21个自由度的仿人灵巧手，其整体框架基于混合驱动理念，将SMA执行器与电机执行器集成于手掌及手指内部。手指关节（除拇指基座）由SMA弹簧驱动，提供弯曲运动；拇指的CMC（腕掌）关节由电机驱动，实现外展/内收和对掌运动；所有手指的DIP（远指间）关节则通过肌腱耦合到PIP（近指间）关节。驱动传输系统包括SMA弹簧、肌腱、滑轮和连杆机构。控制系统负责接收高层指令，并独立控制每个SMA和电机执行器的状态。

核心模块包括：

  1. SMA执行器模块：每个主动手指关节（每个手指的MCP和PIP关节）由一对 antagonistic SMA弹簧驱动。SMA弹簧在加热（通电）时收缩，冷却时伸长。通过交替或同时激活一对弹簧，可实现关节的双向运动与刚度调节。SMA弹簧被紧凑地布置在手指近端指骨和手掌内部。
  2. 电机执行器模块：拇指的CMC关节采用一个安装在手掌内的微型直流电机驱动，通过蜗轮蜗杆减速器传递扭矩，实现拇指的定位。该设计弥补了SMA响应慢的缺点，用于快速调整抓取姿态。
  3. 驱动传输系统：SMA的线性收缩通过滑轮和肌腱转换为关节的旋转运动。设计采用了低摩擦的滑轮和特氟龙涂层肌腱以提高效率。
  4. 控制系统：基于微控制器（STM32）实现。采用脉宽调制（PWM）控制SMA的加热功率，并通过关节处的磁性编码器（AS5048A）进行位置反馈，构成简单的比例（P）控制闭环。电机则由单独的H桥电路驱动。

与现有纯电机或纯SMA驱动的灵巧手相比，其创新点具体体现在：

  • 混合驱动架构：创造性地区分驱动需求，对需要快速定位的拇指基座使用电机，对需要柔顺交互的指间关节使用SMA，在性能与柔顺性间取得平衡。
  • 紧凑的SMA集成：将SMA弹簧直接内置于手指结构中，而非作为外部驱动单元，极大提升了空间利用率和手的集成度。
  • 拮抗式SMA控制：采用成对SMA弹簧驱动单个关节，不仅提供双向驱动能力，还允许通过调节两侧张力来改变关节刚度。

实验与结果

实验平台与数据集：实验将CYJ Hand-0安装在UR5机械臂末端。使用了一系列日常物体（部分来自YCB数据集）进行抓取测试，包括不同形状、大小和重量的物体，如马克杯、玩具、螺丝刀、球体等。此外，还进行了力量测试和动态交互实验。

对比方法：由于是新型驱动架构的原型展示，论文主要将CYJ Hand-0的整体性能与传统电机驱动灵巧手的典型特性（如重量、体积、柔顺性）进行定性对比，并展示了其实现多种抓取模式的能力。

关键实验结果：

  1. 抓取能力：CYJ Hand-0成功演示了对多种物体的稳定抓取，包括精密抓取（握笔）、力量抓取（握水瓶）和包裹式抓取（握球）。实验表明，手部能够适应物体形状，并利用其被动柔顺性稳定抓持。
  2. 力量展示：单个SMA驱动的指节能够提起约100g的重物。整只手可以稳定抓持并移动重量超过500g的物体（如一瓶水）。
  3. 动态交互：展示了手部在受到外部扰动时的柔顺性。当被推动时，手指关节能够被动弯曲吸收冲击，而后通过SMA控制恢复原位。
  4. 响应速度：SMA驱动关节的典型运动周期（如从全伸到全屈）约为1-2秒，证实了其速度较慢；而电机驱动的拇指关节响应迅速。

消融实验总结：论文虽未进行严格的定量消融实验，但通过设计本身和实验演示，实质上对比了“混合驱动”与“纯SMA驱动”概念。结果证明，加入电机驱动的拇指显著提升了手部调整抓取姿态的速度和灵活性，弥补了纯SMA系统响应迟缓的短板，使手能更高效地完成需要快速重定向的抓取任务。

总结与启发

核心贡献：

  1. 提出并实现了一种新颖的SMA-电机混合驱动仿人灵巧手（CYJ Hand-0），为灵巧手设计提供了兼顾柔顺性、功率密度与响应速度的新范式。
  2. 实现了高度集化的机械设计，将21个自由度的驱动系统（包括24个SMA弹簧和1个电机）全部集成于手掌尺寸内，证明了混合驱动在紧凑化方面的潜力。
  3. 通过实验验证了该手在多种抓取任务中的有效性，展示了其柔顺交互、适应不同物体形状的能力。

局限性：

  1. 重量与功耗：由于集成了大量SMA和金属结构，手部自重较大（论文中未明确给出，但暗示是挑战）。SMA持续通电保持位置导致能耗和热管理问题。
  2. SMA性能限制：响应速度慢、循环寿命有限、热致收缩的精确控制仍然具有挑战性。
  3. 控制策略简单：目前仅采用基本的P控制，未充分利用SMA的变刚度特性或实现更高级的力/位混合控制。

对后续研究的启示：

  1. 材料与驱动优化：探索性能更优的SMA材料或新型柔性驱动器，以提升速度、寿命和能效。
  2. 先进控制算法：开发专门针对混合驱动系统和SMA非线性、迟滞特性的先进控制算法，如模型预测控制、强化学习等，以实现更精准、鲁棒的操作。
  3. 系统轻量化：优化机械结构，采用更轻材料，是投入实际应用的关键步骤。
  4. 应用场景拓展：这种具备内在柔顺性的手非常适合人机协作、服务机器人和对安全要求高的环境，值得在这些方向深入探索。
