Dexterous Soft Hands Linearize Feedback-Control for In-Hand Manipulation

arXiv: 2308.10691 作者: Sieler, Adrian, Brock, Oliver 日期: 2023/08/21

📝 详细解读

研究背景与动机

当前，针对灵巧手（尤其是欠驱动、柔顺的软手）进行在手操控的主流方法主要分为两类：基于模型的控制器和强化学习（RL）。基于模型的控制器（如模型预测控制MPC）需要精确的动力学模型，但软体机器人固有的非线性、迟滞和接触动力学使得获取精确模型极其困难。强化学习方法虽然能通过试错学习策略，避开了精确建模的需求，但其样本效率低，训练出的策略通常缺乏鲁棒性，难以适应未见过的新物体或任务。

本文针对的核心痛点是：如何为具有复杂、难以建模动力学的灵巧软手，设计一个简单、高效且通用的反馈控制器，以实现鲁棒的在手操控。论文提出了一个新颖的视角：利用软手自身的物理特性——特别是其柔顺性和欠驱动性——来自然地将原本复杂的非线性系统动力学“线性化”。具体而言，论文认为，当使用低增益的PD控制器跟踪一个虚拟的物体位姿时，软手与物体之间的柔顺接触会使得整个闭环系统（手+物体）的行为近似为一个简单的质量-弹簧-阻尼系统，从而实现稳定的反馈控制。

本文的核心思路可概括为：将灵巧软手本身视为一个反馈控制器，通过设计一个将期望物体运动映射为手指关节角度，再映射为电机位置的框架，利用软手的柔顺特性隐式地实现系统动力学的线性化，从而无需精确模型即可实现鲁棒的在手旋转操控。

方法详解

本文提出的方法框架旨在控制一个多指的、欠驱动的灵巧软手（以论文中使用的Dexterous Soft Hand为例），使其能够通过反馈控制稳定地旋转手中的物体。其核心思想不是为整个系统（手+物体）建立复杂的动力学模型，而是将软手本身作为一个“计算器”和“执行器”，嵌入到一个简单的反馈循环中。

整体框架：控制器输入是期望的物体位姿 ( T_d ) 和当前估计的物体位姿 ( T )，输出是发送给手部各驱动电机的目标位置 ( q_m )。流程分为两步：1）物体运动到手指运动：通过一个抓握雅可比矩阵 ( J_g ) 的伪逆，将物体位姿误差 ( \Delta T ) 映射为达成此物体运动所需的手指关节角度变化 ( \Delta \theta )。2）手指运动到电机运动：通过一个固定的、从关节空间到电机空间的映射矩阵 ( M )，将 ( \Delta \theta ) 转换为电机位置命令 ( \Delta q_m )。然后，一个低增益的PD控制器被用于跟踪这个不断更新的电机位置目标，驱动实际电机。

图1：所提出的反馈控制框架概览。给定期望与当前物体位姿，通过抓握雅可比 (J_g) 的伪逆计算所需的手指关节位移 (\Delta \theta)，再通过线性映射 (M) 转换为电机位移 (\Delta q_m)。一个低增益的PD控制器驱动真实电机跟踪 (q_m)。软手与物体的柔顺接触使整个闭环系统表现为一个线性二阶系统。

核心模块与技术细节：

  1. 抓握雅可比矩阵 (J_g)：这是框架的关键。(J_g) 定义了在特定抓握构型下，手指关节角度变化 ( \Delta \theta ) 与物体位姿变化 ( \Delta T ) 之间的线性关系，即 ( \Delta T = J_g \Delta \theta )。论文的关键假设是，对于软手在稳定抓握一个物体时，(J_g) 可以通过物体的几何形状和接触点估计得到，并且在物体小范围运动期间近似恒定。控制器使用 (J_g^\dagger)（伪逆）从 ( \Delta T ) 求解 ( \Delta \theta )。
  2. 关节到电机的映射 (M)：由于软手是欠驱动的（电机数量少于关节数量），需要通过一个线性映射 (M) 将关节角度目标 ( \theta ) 转换为电机位置目标 ( q_m )，即 ( \theta = M q_m )。这个矩阵 (M) 是固定的，由手的肌腱驱动拓扑结构决定。
  3. 低增益PD控制：对电机位置 ( q_m ) 使用比例-微分（PD）控制，增益设置得较低。这是实现“线性化”的关键。低增益使得电机无法精确、快速地跟踪剧烈变化的 ( q_m )，从而导致手指对物体施加的力是柔顺的。这种柔顺性，加上软材料本身的阻尼，使得物体在手指接触下的运动动力学被“过滤”并简化为一个易于预测和稳定的二阶系统（质量-弹簧-阻尼模型）。

创新点：与需要精确动力学模型的MPC或需要大量数据训练的RL相比，本方法的创新性在于利用机器人本体（软手）的物理特性作为控制器设计的一部分。它不试图克服或精确建模软手的复杂性，而是利用其柔顺性和欠驱动性来简化控制问题，将复杂的非线性接触动力学问题转化为一个简单的、基于几何雅可比的运动学映射问题，并通过低增益控制自然引入稳定化动力学。

实验与结果

实验设置：

  • 机器人平台：Dexterous Soft Hand，一个3D打印、肌腱驱动的五指软手，具有15个关节（每指3个）和5个电机（每指1个）。
  • 感知：使用外部运动捕捉系统（OptiTrack）提供物体（和手基座）的6自由度位姿。
  • 物体：使用YCB数据集中的物体（ mustard bottle, cracker box, banana ）以及一个自定义的、对称性较低的 Smooth Bunny 物体。
  • 对比方法：
    1. 模型预测控制（MPC）：使用基于软手和物体简化动力学模型（忽略接触摩擦细节）的MPC作为性能上限对比。
    2. 强化学习（RL）：使用PPO算法训练一个策略，其输入为物体位姿误差和手部本体感知，输出电机位置增量。
    3. 本文方法（Ours）：即上述反馈控制框架。
    4. 开环基准（Open-Loop）：执行预编程的手指弯曲序列。

    • 展示了控制器能够处理旋转过程中的物体滑动和滚动接触。
    • 展示了在未训练过的、形状更复杂的日常物品（如马克杯、胶水瓶）上也能成功完成旋转操作，体现了良好的泛化能力。

消融实验总结：低增益PD控制是方法成功的关键组件，它确保了系统的柔顺性和稳定性。抓握雅可比矩阵 (J_g) 的近似性可以被容忍，系统的鲁棒性很大程度上源于软手本身的物理特性。

总结与启发

核心贡献：

  1. 提出了一个新颖的反馈控制框架：将灵巧软手本身视为反馈控制器，通过运动学映射（(J_g) 和 (M)）和低增益控制，实现了对复杂软体系统的有效控制，无需精确动力学模型。
  2. 论证并利用了软手的“线性化”特性：从理论和实验上表明，软手的柔顺性与低增益控制相结合，可以使手-物闭环系统表现出近似线性的二阶动力学，这是实现稳定、鲁棒控制的基础。
  3. 展示了卓越的实验性能与泛化能力：在多种物体上的旋转任务成功率超过90%，性能媲美或优于基于简化模型的MPC，并显著优于RL方法，且能泛化到未见过的物体。

局限性（论文提及）：

  • 假设：方法依赖于抓握雅可比矩阵 (J_g) 在操作期间近似恒定，这在物体发生大幅滑动或滚动时可能不成立。
  • 实验范围：目前主要验证了平面内的旋转操作，对于更复杂的6自由度操作或需要频繁重抓握的任务，其性能尚未探索。
  • 感知：实验依赖外部运动捕捉，未来需要集成手部本体感知（如关节角度、肌腱张力）来估计物体位姿和接触状态。

对后续研究的启示：

  • 物理本体作为计算资源：本工作启发了“物理计算”或“形态计算”在机器人控制中的应用，即利用机器人本体（尤其是柔顺本体）的物理特性来简化或替代复杂的数字计算与控制算法。
  • 模型简化新思路：为处理复杂接触动力学的控制问题提供了一条新路径：不是追求更复杂的模型，而是通过巧妙的控制器设计（如低增益）和利用本体特性，主动将系统引导至一个更容易控制的“模式”。
  • 迈向实际应用：该方法简单、高效且鲁棒，为灵巧软手在真实动态环境中完成精细操作（如零件装配、工具使用）提供了有前景的基础控制器，后续可结合视觉伺服和接触状态估计以实现完全自主。
