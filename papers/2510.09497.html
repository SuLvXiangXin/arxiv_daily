Autonomous Soft Robotic Guidewire Navigation via Imitation Learning

arXiv: 2510.09497 作者: Axel Krieger Team 日期: 2025-10-10

📝 详细解读

研究背景与动机

在血管内手术中，介入医生需要将导管和导丝在患者血管内推进至治疗部位。具有机器人尖端可操纵的导丝能增强操控性，但也带来了建模与控制的挑战。目前的主流方法包括基于模型的导航算法、简单的壁面跟随或中心线跟随策略，以及强化学习（RL）或模仿学习（IL）。这些方法存在关键局限性：基于模型的方法对复杂解剖结构计算量大且未结合软体机器人动力学；简单策略需要精确的解剖重建或机器人视觉；RL方法在物理平台上的性能往往显著下降或需要在新解剖结构上重新探索，这在手术环境中不切实际；而现有的IL研究多限于路径规划，未解决灵巧器械操作的困难。本文针对软体机器人导丝在血管内导航的具体痛点——复杂的血管-器械相互作用力、视觉反馈不完整且模糊（仅通过间歇性注射造影剂在X光透视下显影）、以及软体执行器固有的迟滞和非线性动力学，提出了一个端到端模仿学习的新视角。本文核心思路是：开发一个基于Transformer的动作分块模仿学习框架，通过目标条件化、相对动作输出和自动造影剂注射管理，使模型能够泛化到未见过的血管几何结构，实现软体机器人导丝的自主导航。

方法详解

整体框架是一个端到端的策略模型，输入是当前时刻的视觉观测，输出是未来一段时间（一个“块”）的动作序列和一个是否注射造影剂的决策。视觉观测由两部分组成：实时模拟的荧光透视图像和一个静态的目标特征图表示。

图4：自主导航的架构。静态目标表示和实时荧光透视图像被输入到自主策略中。策略输出一个相对动作序列和一个指示是否注射造影剂的二元变量。

核心模块与技术细节：

  1. 目标表示：为了提供精确的导航指引并实现目标条件化，论文设计了一种特征图表示。它由两幅灰度图拼接而成，分别编码图像中每个像素位置到目标动脉瘤中心的x方向距离和y方向距离。血管内部的像素值根据公式 (i - x_a) / c 计算（x方向），血管外像素值设为1。常数c用于归一化。这种表示在每个像素点提供了指向目标的向量场，比单纯的二值化目标图像（仅在动脉瘤区域标记）包含更丰富的导航信息。为了模拟真实手术中血管移动导致的路标图不准确，在训练时会对该目标表示施加一个小的随机刚性变换。
  2. 图像编码与Transformer策略：实时荧光透视图像（单通道）和目标特征图（双通道）分别经过中心裁剪和缩放后，输入两个独立的、在ImageNet上预训练的ResNet-18编码器提取特征。将正弦位置编码添加到编码器输出后，送入一个BERT风格的Transformer编码器-解码器对。解码器输出未来K个时间步的动作“块”。
  3. 动作表示与损失函数：
    • 相对动作输出：模型输出的电机动作（弯曲和推进）是相对于当前电机位置的增量（a_{t+1:t+K}^{motor} - a_t^{motor}）。这种表示有助于克服软体执行器因材料特性和空气可压缩性带来的迟滞，以及平移皮带驱动可能存在的打滑问题，因为它学习的是局部状态变化与动作增量之间的关系，而非不稳定的绝对位置映射。电机动作使用L1损失进行优化。
    • 造影剂注射决策：造影剂注射是一个稀疏事件。为了避免严重的类别不平衡，模型不预测块内每个时间步是否注射，而是预测在整个动作块期间是否至少注射一次。具体通过公式 a_t^{contrast} = σ( max_{s∈[t+1,t+K]} ĉ_s ) 实现，其中ĉ_s是每个时间步的二元预测，σ是sigmoid函数。该输出使用加权二元交叉熵损失（正类权重为7）进行优化。总损失为 L = l_{L1} + 0.5 * l_{bce}。
  4. 数据收集与DAgger：训练数据包含在36个不同迷宫几何结构上收集的演示。初始收集了218条“正常”演示（从迷宫起点开始）和142条“恢复”演示（从预期失败模式开始）。随后采用DAgger方法进行数据聚合：用初始模型在训练迷宫中评估，当模型失败时，由操作员接管并引导至目标，记录接管后的序列作为新的恢复演示，共添加287条。最终训练集总计647条演示（218条正常，429条恢复）。

创新点：1）目标条件化的特征图表示，提供了密集的导航信号，优于简单的目标图像掩码。2）相对动作输出，适应了软体机器人的非线性动力学和不确定性。3）智能造影剂管理，模型学会在需要定位更新时才触发注射，而不是固定频率或由人类决定，这更符合临床限制（造影剂有毒）。

实验与结果

实验设置：使用自定义的大型3D打印软体气动机器人平台和模拟荧光透视系统。评估在两个测试集上进行，均未出现在训练集中：1) 重排几何结构：3个迷宫，由训练集中出现过的模块（入口、分叉、分支）以新组合构成。2) 新颖几何结构：3个迷宫，包含训练集中完全未出现过的分叉和分支模块（见图3）。每个目标动脉瘤尝试3次试验，成功定义为机器人在60秒内使尖端进入目标动脉瘤区域。

图3：训练集、重排测试集和新颖测试集的迷宫设计示意图。重排集使用训练集中见过的模块进行新组合，新颖集则使用全新的模块。

关键参数确定：首先确定了最佳动作块大小为2秒（对应50个时间步），其在重排几何结构上的成功率为89%，优于1秒（28%）、3秒（61%）和4秒（78%）的块大小。

消融实验：对框架的四个关键设计进行了消融研究，结果如下表（成功率）及图5（最终距离分布）所示。

图5：各消融模型试验结束时，机器人尖端到目标动脉瘤边界的距离分布。每个模型在3个重排和3个新颖几何结构上，每个目标动脉瘤评估3次，每个条形代表18次试验。

  • 恢复数据：完全不用恢复数据（0%恢复）的模型性能大幅下降（新颖集成功率22%）。仅使用一半恢复数据（50%恢复）的模型在重排集上与完整模型持平（89%），但在更分布外的新颖集上成功率降至61%，且平均最终距离更远（9.6 mm vs 0.6 mm），表明充足的恢复数据对泛化至全新几何结构至关重要。
  • 造影剂注射：与固定间隔注射（每8秒或16秒）的基线相比，模型预测注射的方式在新颖集上成功率显著更高（83% vs 50-56%）。有趣的是，更频繁的固定注射（每8秒）性能反而更差，作者推测模型可能将注射与“恢复”行为关联，导致过度使用恢复策略。
  • 目标表示：使用二值目标掩码（“Binary goal”）的模型成功率（新颖集61%）低于提出的特征图表示（83%）。虽然其平均最终距离（2.0 mm）尚可，但常无法执行最后的关键动作使尖端进入目标。没有目标信息的模型（“No goal”）性能最差。
  • 动作表示：使用绝对动作输出的模型性能下降（新颖集成功率39% vs 83%），且在从重排集到新颖集时性能下降幅度最大（33%），表明相对动作表示对应对新环境动力学变化的重要性。

提出的模型取得了83%的成功率（15/18），与两位神经介入专家医师（83%和78%）的表现相当，并显著优于扩散策略（22%）、多层感知机（28%）和经典的中心线跟随控制器（50%）。

总结与启发

核心贡献：1) 提出了首个用于软体机器人导丝导航的端到端模仿学习算法。2) 引入了目标条件化的特征图表示、相对动作输出和模型控制的自动造影剂注射这三个关键设计，有效提升了在模拟荧光透视成像下的导航性能和泛化能力。3) 通过系统的消融实验在物理平台上验证了每个设计选择对于泛化到未见几何结构的重要性。

局限性：论文明确指出，研究使用了一个大型机器人原型，并在投影到2D平面的简化血管分叉环境中进行。这虽然保留了核心挑战（如软体动力学、视觉约束），但与真实临床场景（3D、毫米级尺度、更复杂的血管网络）仍有距离。

后续研究启示：这项工作证明了端到端模仿学习在处理复杂软体机器人控制问题上的潜力，特别是在存在严重不确定性和间歇性感知的场景下。所提出的目标表示、相对动作和智能感知触发机制，为其他类似领域的机器人控制（如其他腔内手术或柔性机器人操作）提供了可借鉴的设计思路。未来的工作自然是将该方法扩展到更小尺度、3D的解剖结构以及更复杂的导航任务中。

💡 一句话总结

本论文致力于解决软体机器人导丝在血管内手术中自动化导航的建模与控制难题。其核心技术是开发了一个基于Transformer的模仿学习框架，该框架整合了目标条件设定、相对动作输出与自动对比剂注射。通过在36种分叉血管几何结构上进行训练，并在3种未见过的血管结构上评估，该模型能够自主将导丝尖端导航至动脉瘤目标位置，成功率达到83%，优于多个基线方法。
