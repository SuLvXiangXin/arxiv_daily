Intelligent Singularity Avoidance in UR10 Robotic Arm Path Planning Using Hybrid Fuzzy Logic and Reinforcement Learning

arXiv: 2601.05836 作者: Jyh-Horng Wu Team 日期: 2026-01-09

📝 详细解读

研究背景与动机

在机器人操作中，机械臂运行至奇异性构型附近时，会面临可操作性丧失、关节速度无限增大以及系统失稳等严峻挑战。当前主流方法主要依赖于数学手段，如阻尼最小二乘法、奇异性鲁棒逆运动学以及工作空间分解技术。这些方法虽然有效，但通常缺乏对不同操作条件的适应性，并且其规避策略可能过于保守，限制了机器人的有效工作空间。本文针对机械臂路径规划中的奇异性规避问题，旨在解决实时奇异性检测、围绕奇异区域的智能路径规划以及从操作经验中自适应学习三大挑战。本文提出了一种新颖的混合方法，其核心思路是：将基于多种指标的实时奇异性检测、一个用于安全决策的模糊逻辑系统，以及一个用于自适应路径规划的稳定强化学习框架相结合，以实现整个工作空间内的安全操作。

方法详解

本文提出的系统架构集成了四个主要组件：奇异性检测引擎、模糊逻辑安全系统、强化学习路径规划器以及仿真/硬件接口，形成一个模块化设计，通过多重冗余检查确保系统安全。

图1：系统整体架构。展示了奇异性检测引擎、模糊逻辑安全系统、强化学习（RL）路径规划器与PyBullet仿真及URSim硬件接口的集成关系。

奇异性检测引擎 采用三种互补的指标进行全面分析：

  1. 可操作性度量 μ：量化机器人在所有方向上运动和施加力的能力，计算公式为 μ = √det(J(q)J(q)ᵀ)，其中 J(q) 为关节构型 q 对应的雅可比矩阵。
  2. 条件数 κ：反映雅可比矩阵的数值条件，在接近奇异性时趋于无穷，计算公式为 κ = σ_max / σ_min，其中 σ_max 和 σ_min 分别为最大和最小奇异值。
  3. 最小奇异值 σ_min：直接反映与奇异构型的接近程度，通过对雅可比矩阵进行奇异值分解 J = UΣVᵀ 获得。

模糊逻辑安全系统 实时处理奇异性指标及关节速度，以做出安全决策。系统输入变量包括可操作性水平、条件数质量和关节速度大小，每个变量设有五个语言项（如“非常低”、“低”、“中等”等）。输出为安全等级分类，从需要立即停止的“紧急停止”到操作条件极佳的“最优”。该系统采用了三角隶属度函数，并基于45条精心设计的规则进行推理，规则权重根据安全关键性分配（例如，紧急停止规则权重为1.0）。

强化学习框架 采用经过稳定性增强的近端策略优化（PPO）算法来学习自适应路径规划策略。状态空间包括关节位置、目标坐标和奇异性度量，动作代表关节速度指令。奖励函数平衡了多个目标：R_t = R_distance + R_success + R_progress - P_singularity - P_velocity，其中包含基于距离的奖励、目标达成奖励、渐进奖励，以及对接近奇异性和关节速度过快的惩罚。策略网络和价值网络采用保守架构，包含层归一化和有界激活函数以防止训练发散，并使用了梯度裁剪和自适应学习率以确保稳定收敛。

集成与工作流程：系统使用PyBullet仿真和官方UR10模型进行安全的训练数据收集。运动学引擎利用 scipy.optimize 实现数值逆运动学，为给定目标位置生成多个解，并优先选择模糊逻辑系统安全评分高的解。RL训练采用课程学习策略，分四个难度递增的阶段进行，每个阶段需持续达到设定的成功率阈值（如60%，70%，80%，85%）才能进入下一阶段。实时安全监控以10Hz频率运行，集成了软件和硬件层面的急停功能。

总结与启发

本文的核心贡献包括：1) 一个综合的奇异性检测框架，同时利用可操作性度量、条件数和最小奇异值三种互补指标；2) 一个包含45条优化规则的模糊逻辑安全系统，用于实时、稳健的安全决策；3) 一个具有稳定收敛特性的强化学习框架，通过课程学习实现自适应路径规划；4) 实验验证表明系统能达到90%的成功率并始终保持安全性。

论文自身提到的局限性包括：实时RL推理的计算要求较高、初始训练需要大量仿真时间、系统依赖准确的关节位置反馈。

这项工作为后续研究提供了重要启示：首先，这种混合架构（模糊逻辑提供即时安全保证，RL提供长期优化）可扩展到更复杂的场景，如多机械臂协调系统或动态移动障碍物的避障。其次，为满足实时性要求，开发嵌入式或轻量化实现是未来的一个方向。最后，该方法展示了如何将数据驱动的学习与传统基于模型的控制逻辑有效结合，为机器人安全控制提供了新思路。

💡 一句话总结

本文针对UR10机械臂路径规划中的奇异性避免问题，提出混合模糊逻辑与强化学习的智能系统，解决奇异性导致控制丢失和设备损坏的核心挑战。方法整合实时奇异性检测（基于可操作性度量和条件数分析）、模糊逻辑决策以及强化学习自适应路径规划。实验表明，系统在保持与奇异性配置安全距离的同时，到达目标位置的成功率达到90%。
