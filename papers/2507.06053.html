SCCRUB: Surface Cleaning Compliant Robot Utilizing Bristles

arXiv: 2507.06053 作者: Jeffrey Ian Lipton Team 日期: 2025-07-08

📝 详细解读

研究背景与动机

目前，在人类居住环境中执行清洁任务的机器人主要面临安全性与有效性的矛盾。刚性机械臂虽然能施加清洁顽固污渍所需的大力和扭矩，但其高刚度和惯性带来了安全风险，通常被限制在与人隔离的结构化环境中。相反，软体机械臂因其顺应性可以安全地在人类周围工作并适应环境不确定性，但其通常难以提供有效擦洗所需的持续扭矩或大侧向力。本文针对“软体机械臂难以提供持续扭矩进行擦洗”这一具体痛点，提出了一种新视角：利用具有高“扭转-弯曲”比的软体臂，使其能够传输扭矩来驱动旋转刷头，通过压力和旋转产生的摩擦力（而非侧向力）进行清洁。本文的核心思路是：设计一个能显著减少反作用力矩的对转刷头，并结合一个学习软体臂在负载下逆运动学与弹性行为的神经网络模型，实现开环的力与位置控制，从而让软体机器人安全且有效地擦洗表面顽固污渍。

方法详解

本文方法的整体框架基于一个能够传输连续扭矩的软体机械臂（TRUNC臂），在其末端集成一个专门设计的对转刷头，并通过一个经过训练的神经网络模型进行轨迹规划和力控制。

图1：软体擦洗机器人概览。(A) 通过肌腱驱动的机械臂（L=710 mm），末端安装刷头和配重。基座电机通过贯穿臂体的同心轴传输扭矩驱动刷子。(B) 臂体的顺应性使其能够安全地进行高接触力的表面交互。

核心模块一：对转刷头设计与建模。为了在软体臂末端实现旋转擦洗并最小化反作用力矩对柔性臂体稳定性的影响，本文设计了一个基于行星齿轮箱的对转刷头。它将单一的输入旋转（ω = 110 RPM）分解为两个相反方向的旋转。

图3：(A) 基于行星齿轮箱的对转刷头设计。(B) 实验表明，对转刷头产生的净扭矩显著低于单向刷。误差条代表 n=5 次试验的最小值和最大值。

其理论基础是对旋转刷产生的摩擦力矩进行建模。如图2所示，在低平移速度下，摩擦力主要由刷子的角速度ω主导，可简化为作用在刷子-表面界面的净力矩M。对于一个半径为r、承受均匀压力P的刷子，净力矩为 M = (2πμP r³)/3。对于对转刷头，内刷面（实心圆盘，半径ri）和外刷面（环形，外半径ro）旋转方向相反，其净力矩为 M_net = (2πμP)/3 (ro³ - 2ri³)。通过设置 M_net = 0，可推导出实现零净力矩的理论半径比为 ro/ri = 2^(1/3) ≈ 1.26（即 ri ≈ 79% ro）。实际设计中，受刷毛间距限制，最终采用 ro=35 mm, ri=25 mm（比例1.4），略偏离理想值。实验验证表明，该设计将擦洗时的净反作用力矩平均降低了85%。

图2：(A) 将擦洗建模为一个在均匀压力P下，以角速度ω旋转、并以速度v平移的刷子。(B) 对于低平移速度，刷毛与表面之间的动摩擦被建模为作用在刷子上的净力矩。

核心模块二：负载下的臂体建模与神经网络学习。擦洗需要在对表面施加持续压力的同时保持工具的位置控制。外部负载会导致软体末端执行器发生显著形变。本文扩展了先前的学习框架，以考虑外部加载下的臂体弹性行为。

图4：(A) 使用九根肌腱长度 li 定义臂体的配置空间，这些肌腱连接在臂体的三个阶段 s 上。(B) 假设刷子比臂体主体重，将其建模为作用于末端执行器的点载荷 Fg。

将臂体的配置空间定义为九根肌腱的长度 q ∈ ℝ^9，末端执行器的位姿定义为 x = [x, y, z, qw, qx, qy, qz]。在准静态平衡假设下，内部肌腱张力 Ft 可由已知的末端重力载荷 Fg 和作为控制输入的法向力 Fn 计算：Ft = Fg - Fn。目标是学习一个增强的逆运动学映射 f⁻¹: (x, Ft) ↦ q，该映射将臂体的运动学和弹性编码到一个单一函数中。

为此，收集了一个包含10,000个样本的力-形变数据集，其中末端重力载荷 Fg 从6.2 N变化到15.3 N。

图5：用于训练神经网络的力-形变数据集（10,000个样本），从(A)顶部和(B)侧视图。每种颜色代表图例中指示的不同 Fg 加载条件。

使用该数据集训练一个全连接神经网络（两层隐藏层，每层1600个神经元，ReLU激活）。训练后的模型能够补偿外部负载引起的形变。实验验证中，让臂体在悬挂3.4 N配重的情况下跟踪圆形轨迹，新模型将平均位置误差从旧模型的15.7 mm (2.21% L) 降低到7.0 mm (0.98% L)，平均方向误差从2.1°降低到1.3°。

图6：使用神经网络模型生成圆形轨迹供臂体跟踪。末端执行器附加了3.4 N配重以产生外部负载。(A) 使用原始网络，臂体向内偏转并跟踪一个更小的圆。(B) 新模型补偿了这种偏转，使臂体更准确地跟踪圆形。

核心模块三：开环力控制。基于学习到的模型和静态平衡方程，可以实现开环力控制。对于期望的末端位姿 x 和法向力 Fn，先计算所需的肌腱张力 Ft = Fg - Fn，然后将 (x, Ft) 输入神经网络，推理得到对应的电机输入 q。实验表明，通过设置 Fg = 15.3 N，臂体可以将接触力从1 N调制到8.5 N，并能在不同径向位置（Δr 从 0 到 75 mm）实现力控制，尽管存在增益误差和约0.43 N的标准偏差波动。

图7：(A) 展示了在径向距离扫描范围内的开环力控制。(B) 显示臂体可以将接触力从1 N调制到8.5 N，并紧密跟随目标线性斜坡。阴影误差条代表 n=5 次试验的最小值和最大值。

与现有方法相比，本文的创新点具体体现在：1) 首次设计并验证了用于软体机械臂、能大幅降低反作用力矩的对转刷头末端执行器；2) 将软体臂的逆运动学学习框架扩展至包含可变外部负载的情况，实现了同时补偿形变和进行力控制。

实验与结果

本文使用了多个实验平台和数据集进行验证：

  1. 刷头力矩测试：使用Instron机器和±5 N·m双轴载荷传感器，在60目砂纸表面测量刷头的反作用力矩与法向力的关系。
  2. 臂体建模数据集：使用运动捕捉系统，在末端加载不同配重（Fg 从6.2 N到15.3 N）下，收集了10,000个臂体位姿与肌腱长度的对应关系数据。
  3. 力控制验证：使用安装在表面的称重传感器测量实际接触力。
  4. 清洁性能测试：在真实的玻璃盘和儿科马桶座圈表面进行，污染物为烧焦的番茄酱和粘性果酱。

对比的Baseline包括：1) 单向刷头（作为力矩对比）；2) 仅湿润和冲洗（作为清洁效果对比）。

关键实验结果：

  1. 刷头性能：对转刷头将擦洗时的净反作用力矩平均降低了**85%**。实验测得的力矩-力关系斜率（9.96 N·mm/N）与理论预测（9.34 N·mm/N）接近。
  2. 模型精度：新训练的神经网络模型在跟踪带载轨迹时，达到了7.0 mm（臂长的0.98%）的平均位置误差和1.3° 的平均方向误差。
  3. 力控制：臂体能够实现1 N至8.5 N的开环力控制。在不同径向位置，力控的R²值在0.923到0.995之间，但存在增益误差（8.85% 到 35.7%）。
  4. 清洁效果：机器人成功清除了玻璃盘和马桶座圈上的顽固污渍。与仅进行湿润和冲洗（平均去除率**47.1%）相比，机器人擦洗后的平均污染去除率达到99.7%**。

图8：使用机械臂擦洗玻璃盘和马桶座圈。人工事先湿润表面，并在机械臂完成后冲洗一次。分别使用烧焦的番茄酱和果酱模拟盘子和座圈上的污染物。前后对比图像显示刷子有效去除了污染物。

消融实验主要体现在模型对比上：使用旧模型（仅在单一负载下训练）跟踪带载轨迹，位置误差为15.7 mm；而使用新模型（在多种负载下训练）后，误差降至7.0 mm。这证明了扩展数据集以涵盖不同负载条件对于补偿形变、实现精确轨迹跟踪的必要性和有效性。

总结与启发

本文的核心贡献可概括为三点：

  1. 展示了软体机械臂执行有效擦洗的能力：首次实现了一个软体机器人系统，能够安全地擦除附着在常见家庭表面（如盘子、马桶座圈）上的顽固污染物，平均去除率高达99.7%。
  2. 扩展了软体扭矩传输臂的控制框架：通过收集力-形变数据集并训练神经网络，将逆运动学学习框架扩展到能够解释和补偿由可变外部负载引起的大幅偏转，从而实现了开环的力与位置控制。
  3. 设计了最小化反作用力矩的末端执行器：创新性地设计并验证了一个对转刷头，通过行星齿轮结构使内外刷面反向旋转，将擦洗时作用在柔性臂体上的净扭矩平均降低了85%，显著提高了系统稳定性。

论文自身提到的局限性包括：

  1. 受刷毛间距限制，实际制造的对转刷头尺寸（ro/ri=1.4）与理论最优值（1.26）存在11.1% 的误差。
  2. 开环力控制存在增益误差和力波动（标准偏差0.43 N），表明模型在泛化到训练数据分布之外的精确力控方面仍有提升空间。
  3. 当前工作假设刷子在平面上移动且平移速度较低，模型未考虑更复杂的表面几何形状或高动态摩擦效应。

对后续研究的启示：

  1. 可以进一步优化刷头设计，例如使用更密集或可定制的刷毛布局，以更接近理论最优的力矩抵消比例。
  2. 探索闭环力反馈控制或更高级的模型（如考虑摩擦动态特性）以提高力控精度和鲁棒性。
  3. 将该方法扩展到非平面、曲率更大的表面清洁任务，验证软体臂在更复杂家庭环境中的适应性和清洁效能。

💡 一句话总结

本文提出柔性清洁机器人SCCRUB，旨在解决传统刚性机械臂在人类共存环境中清洁时存在安全风险，而柔性机械臂又难以提供持续扭矩或侧向力以清除顽固污渍的核心问题。关键技术包括：1）采用肌腱驱动柔性臂，并在末端安装刷头，通过同心轴结构传递扭矩；2）训练神经网络学习机械臂的逆运动学与弹性特性，实现开环力与位置控制。实验表明，该机器人能安全有效地清除盘子上的烧焦残渣和马桶座上的果酱，平均污染物清除率达到99.7%。
