研究背景与动机

当前，高保真物理模拟是规模化机器人学习的关键，但模拟与真实世界之间的差距（sim-to-real gap）依然显著，尤其是在涉及复杂、动态且不连续的物理接触任务中。主流的缓解方法是显式系统识别，即通过调整模拟器的几何、摩擦、质量等物理参数，使模拟轨迹匹配真实轨迹。然而，这种方法本质上假设一个低维的参数化校正就足够，对于接触丰富的灵巧操作任务而言并不充分，因为许多误差源是高维、状态相关或源于模拟的离散积分。此外，侧重于视觉的模拟到真实技术（如域随机化）主要解决了感知差距，但底层接触动力学模型并未改变。并行发展的隐式对齐方法（如神经残差模型）常将接触不连续性视为噪声，或仅依赖运动学信号，未能充分利用操作中可用的最丰富信号之一——接触信息本身。

本文针对接触丰富操作任务中模拟与真实动力学难以对齐的核心痛点，提出了一个新颖的视角：利用来自触觉传感器的接触信息进行隐式对齐。本文的核心思路是：首先利用大规模模拟数据训练一个基础的神经动力学模型，使其学习从状态、动作和接触历史中预测接触诱导的物理行为；然后，使用包含触觉信息的小批量真实世界交互数据对该模型进行微调，从而更好地捕捉真实接触模式，最终得到一个在模拟和真实世界中表现一致的接触感知神经动力学模型。

方法详解

本文提出的方法是一个隐式模拟到真实对齐框架，其整体流程如图1所示。该流程分为两个阶段：第一阶段，利用大规模模拟数据训练一个基础的神经动力学模型；第二阶段，使用少量包含触觉信息的真实世界交互数据对该模型进行微调，从而获得接触感知的神经动力学模型。

图1：隐式模拟到真实对齐框架。左侧为大规模模拟数据训练基础神经动力学模型；右侧为使用少量真实世界交互数据（含触觉信息）进行微调，最终得到接触感知的神经动力学模型。

模型的具体架构如图2所示，由两个耦合模块组成：(i) 接触预测器模块，用于预测未来的接触概率；(ii) 扩散姿态预测器模块，在给定状态、动作历史及预测的接触信息条件下，生成未来的姿态变化。

图2：接触感知神经动力学框架概览。模型输入为多模态历史H_t，包括过去的目标姿态、关节值、机器人动作、二进制接触信号以及目标点云。通过时序编码器和PointNet编码器提取特征并融合为潜在表示z_t。接触预测模块基于z_t推断未来接触序列，该序列被编码为接触特征f_c，并与z_t拼接形成条件向量h_t，用于条件扩散姿态预测器输出未来姿态增量。

整体输入与目标：在每个时间步t，模型输入为一个固定长度的观测历史窗口H_t，包含过去K+1步的目标姿态s、机器人动作a、关节配置q、二进制接触状态c以及静态目标点云P。模型的目标是预测未来H步的接触序列和姿态轨迹：ˆc_{t+1:t+H} = f_φ(H_t)， Δˆ s_{t+1:t+H} = g_θ(H_t, ˆc_{t+1:t+H})。

核心模块与技术细节：

  1. 多模态编码与融合：时序历史信息（姿态、动作、关节值）沿历史维度堆叠输入网络，接触序列由独立模块编码，静态点云P由PointNet编码器处理。所有模态的嵌入被拼接并通过一个轻量级MLP融合，得到共享的潜在特征z_t ∈ R^512。
  2. 阶段 I: 接触预测器：一个MLP以z_t为输入，预测H步的接触概率序列ˆc_{t+1:t+H}，使用二元交叉熵（BCE）损失L_cnt进行监督。预测的概率序列通过另一个小MLP投影为紧凑的接触特征f_c ∈ R^64。最终，条件向量通过拼接得到：h_t = [z_t; f_c]。
  3. 阶段 II: 扩散姿态预测器：未来姿态变化x_0 = Δs_{t+1:t+H}（表示为6D形式）的分布通过条件去噪扩散模型建模。前向过程定义为q(x_t | x_0) = N(√ᾱ_t x_0, (1-ᾱ_t)I)。反向去噪过程由一个1D U-Net参数化：ε_θ = UNet_1D(x_t, t, h_t)，其中h_t通过FiLM调制应用于所有层。训练目标为L_diff = E[||ε - ε_θ(x_t, t, h_t)||^2_2]。最终轨迹通过将恢复的增量姿态ˆ x_0应用于当前姿态s_t来重建。
  4. 联合目标：整体训练损失为L = L_cnt + λL_diff，联合优化接触预测和接触条件下的动力学。

创新点：

  • 接触感知的表示：明确地将二进制接触预测作为动力学预测的条件信号，使模型能够感知并利用接触事件的离散结构信息。
  • 两阶段训练框架：先利用大规模模拟数据预训练模型作为强先验，再用少量真实数据微调，实现了数据效率和高保真度的平衡。
  • 隐式对齐：通过在多模态潜在表示中共同训练模拟和真实数据，隐式地对齐了模拟与真实世界的状态分布，而非显式调整模拟器参数。

实验与结果

实验设置：

  • 平台与数据集：真实世界实验使用搭载XHand（带力触觉传感器）的XArm7机械臂，操作对象来自YCB数据集。使用FoundationPose进行目标姿态估计。模拟实验在MuJoCo中进行，使用了单个对象（芥末瓶，8000条轨迹）和多个对象（40个YCB对象，15000条轨迹）的设置。
  • 对比方法：对比了多个神经动力学基线模型，包括MLP、UNet、Diffusion-UNet以及一个未使用接触信息的Diffusion-UNet变体。还引用了一个基线方法[52]。
  • 评估指标：使用均方误差（MSE，越低越好）和ADD-S曲线下面积（AUC，简称ADD-S，越高越好）来量化长时域预测精度。ADD-S反映了预测轨迹在3D空间中保持目标几何一致性的能力。

关键实验结果：
表1展示了单对象和多对象任务在不同训练机制下的定量比较结果。在所有设置下，带有接触条件的Diffusion-UNet（即本文方法）均取得了最佳性能。特别是在“模拟+真实”协同训练设置下，本文方法在单对象任务上取得了0.0082的MSE和88.23%的ADD-S；在多对象任务上取得了0.0058的MSE和79.12%的ADD-S，显著优于其他基线。

图5：多步轨迹展开对比。左图显示了本文两阶段接触感知模型预测与真实值的对比；右图显示了直接动力学预测器的展开结果与相同真实值的对比。本文方法通过接触条件化，保持了空间一致性并建模了接触驱动的不连续性，而忽略接触的单步预测器则因误差累积产生显著漂移。

表2展示了任务成功率（预测轨迹终点与真实终点偏差小于5cm的百分比）。在协同训练（Sim+Real）设置下，成功率相比仅使用真实数据训练（Real-only）有显著提升，单对象任务从52.6%提升至73.7%，多对象任务从47.1%提升至64.7%。

消融实验分析：表1本身包含了消融研究。对比“Diffusion-UNet”和“Diffusion-UNet w/ Contact”两行数据可以看出，在几乎所有实验设置下，引入接触条件都带来了MSE的降低和ADD-S的提升，这直接证明了接触感知模块的有效性和必要性。例如，在单对象协同训练设置下，加入接触条件使MSE从0.0091降至0.0082，ADD-S从82.45%提升至88.23%。

总结与启发

核心贡献：

  1. 提出了一个接触感知的神经动力学模型，首次明确地将二进制接触预测作为条件信号集成到前向动力学建模中，以捕捉接触丰富的操作任务中固有的非平滑不连续性。
  2. 设计了一个两阶段、隐式的模拟到真实对齐框架：先利用大规模模拟数据预训练模型作为强先验，再使用少量真实触觉数据进行微调，实现了数据高效且高保真的动力学对齐。
  3. 通过系统的实验验证，证明了该方法在长时域预测精度、模拟到真实迁移以及任务成功率上均优于现有的神经动力学基线，为接触丰富操作的模拟到真实转移提供了一个可扩展的数据驱动方案。

局限性：论文自身提及的局限性包括：方法依赖于机器人硬件上的触觉传感器来获取真实的接触信号；采用的二进制、手级别的接触表示虽然鲁棒，但可能丢失了更细粒度的接触力分布信息，这些信息对于某些精细操作可能很重要。

对后续研究的启示：

  • 接触信息的关键性：这项工作强有力地证明了接触信息（即使是二进制的）对于对齐模拟与真实动力学、提升预测物理真实性的巨大价值。后续研究可以探索更丰富的接触表示（如接触力/力矩、接触点分布）。
  • 混合训练策略的有效性：“大规模模拟预训练 + 小批量真实微调”的范式在机器人学习领域具有普遍意义，能有效结合模拟的规模性与真实世界的保真度。
  • 扩散模型的应用：展示了扩散模型在生成复杂、条件化的机器人状态序列（如物体运动轨迹）方面的潜力，这为动力学建模和控制策略优化提供了新的工具。

💡 一句话总结

本文针对机器人操作中仿真与现实（sim-to-real）的差距问题，特别是在接触丰富的任务中，传统显式系统识别方法难以对齐高维、非连续动态的局限性。提出了一种隐式仿真-现实对齐框架，核心方法是**接触感知神经动力学模型**：先利用大规模仿真数据预训练基础模型，学习状态、动作与接触历史；再通过少量含触觉信息的真实交互数据微调，使模型精准捕获真实接触模式。实验表明，该学习到的前向动力学模型显著提升了状态预测准确性，并能有效用于预测策略性能、优化纯仿真训练的策略，为实现可扩展的数据驱动仿真-现实对齐提供了途径。
